给定一个 `N 叉树`，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

`N 叉树`输入按层序遍历序列化表示，每组子节点由空值分隔（请参见示例）。

- 输入：root = [1,null,3,2,4,null,5,6]
- 输出：3

- 输出：5

---

`N叉树Node的构造`

```java
// Definition for a Node.
class Node {
    public int val;
    public List<Node> children;

    public Node() {}

    public Node(int _val) {
        val = _val;
    }

    public Node(int _val, List<Node> _children) {
        val = _val;
        children = _children;
    }
};
```

---
`java语法`
```java
class Solution {
    public int maxDepth(Node root) {
        Queue<Node> que = new LinkedList<>();
        if(root == null)return 0;
        int nodeNum = 0;
        que.offer(root);
        while(!que.isEmpty()){
            int LevelSize = que.size();
            // 层序遍历算法，LevelSize一层一层遍历
            for(int i = 0; i < LevelSize; i++){
                Node node = que.poll();
                for(int j = 0; j < node.children.size(); j++){
                    que.offer(node.children.get(j));
                }
            }
            nodeNum++;
        }
        return nodeNum;
    }
}
```
`python语法`
```python
class Solution(object):
    def maxDepth(self, root):
        """
        :type root: Node
        :rtype: int
        """
        queue = [] # 设置队列用于遍历
        node_num = 0
        if(root == None):
            return node_num
        queue.append(root)
        while(len(queue) != 0):
            level_size = len(queue)
            node_num += 1 # 层数+1
            for i in range(level_size): # 将这一层的所有node弹出
                node = queue.pop(0)
                # 按children列表size的大小append
                for i in range(len(node.children)):
                    queue.append(node.children[i])
        return node_num
```
