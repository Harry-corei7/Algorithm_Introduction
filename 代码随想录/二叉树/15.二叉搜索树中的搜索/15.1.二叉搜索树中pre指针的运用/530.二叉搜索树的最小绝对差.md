- 给你一个二叉搜索树的根节点`root`，返回 树中任意两不同节点值之间的最小差值 。

- 差值是一个正数，其数值等于两值之差的绝对值。

---

`python语法`
```python
class Solution(object):
    def getMinimumDifference(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        stack = []
        pre = None
        result = sys.maxsize
        if(root == None): return False
        stack.append(root)
        while(len(stack) != 0):
            node = stack.pop(-1)
            if(node != None): # 中序遍历: 左中右，入栈: 右中左
                if(node.right != None): stack.append(node.right)
                stack.append(node)
                stack.append(None)
                if(node.left != None): stack.append(node.left)
            else:
                temp = stack.pop(-1)
                if(pre != None):
                    if(result > (temp.val-pre.val)): result = (temp.val-pre.val)
                pre = temp
        return result
```
