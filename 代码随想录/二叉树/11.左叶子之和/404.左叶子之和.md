给定二叉树的根节点 root ，返回所有左叶子之和。

1) 示例 1：

<img width="277" height="302" alt="image" src="https://github.com/user-attachments/assets/dc97beb3-5603-4b02-b2ce-e71a55a196b6" />

- 输入: root = [3,9,20,null,null,15,7] 
- 输出: 24 
- 解释: 在这个二叉树中，有两个左叶子，分别是 9 和 15，所以返回 24


2) 示例 2:
- 输入: root = [1]
- 输出: 0

---
`python语法`
```python
class Solution(object):
    def sumOfLeftLeaves(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        queue = [] # 设置队列用于遍历
        result = 0 # 设置答案result
        if(root == None): return result
        queue.append(root)
        while(len(queue) != 0):
            node = queue.pop(0) # 将"当前"节点弹出
            # 根据"父节点"判断子节点: 将"当前"节点作为父节点，判断左子节点是否为"左叶子"
            if(node.left != None and node.left.left == None and node.left.right == None):
                result += node.left.val
            # 新的节点入队列，用于遍历
            if(node.left != None): queue.append(node.left)
            if(node.right != None): queue.append(node.right)
        return result
```
