`难度: 中等`

- 给定一个二叉树的 根节点 root，请找出该二叉树的 最底层 最左边 节点的值。
- 假设二叉树中至少有一个节点。

- 输入: root = [2,1,3]
- 输出: 1

- 输入: [1,2,3,4,null,5,6,null,null,7]
- 输出: 7

---
`python语法`
```python
class Solution(object):
    def findBottomLeftValue(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        queue = [] # 设置队列用于遍历
        res_left_nodes = [] # 设置列表收集每一层的第一个左节点(这样直接返回[-1]即为最后一层的最左)
        if(root == None): return 0
        queue.append(root)
        while(len(queue) != 0):
            level_size = len(queue) # 记录这一层的节点数量
            for i in range(level_size):
                node = queue.pop(0)
                if(i == 0): res_left_nodes.append(node.val) # 如果是本层第一个节点(最左节点)，记录val
                # 新的一层节点入队列，注意顺序(先左后右)
                if(node.left != None): queue.append(node.left)
                if(node.right != None): queue.append(node.right)
        return res_left_nodes[-1]
```
