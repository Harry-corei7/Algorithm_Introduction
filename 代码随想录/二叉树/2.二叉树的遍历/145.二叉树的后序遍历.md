`1.递归遍历`

```java
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        postorder(root, res);
        return res;
    }
    public void postorder(TreeNode node, List<Integer> res){
        if(node == null){
            return;
        }
        postorder(node.left, res);
        postorder(node.right, res);
        res.add(node.val);
    }
}
```

`二叉树案例`

```
        1
       / \
      2   3
     / \
    4   5
```
