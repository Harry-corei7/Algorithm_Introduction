`难度：中等`

- 你是一个专业的小偷，计划偷窃沿街的房屋。
- 每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，
  - 如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。
- 给定一个代表每个房屋存放金额的非负整数数组，计算你**不触动警报装置的情况下**，一夜之内能够偷窃到的最高金额。

- 示例 1：
  - 输入：[1,2,3,1]
  - 输出：4
    - 解释：偷窃 1 号房屋 (金额 = 1) ，然后偷窃 3 号房屋 (金额 = 3)。
    - 偷窃到的最高金额 = 1 + 3 = 4 。
- 示例 2：
  - 输入：[2,7,9,3,1]
  - 输出：12
    - 解释：偷窃 1 号房屋 (金额 = 2), 偷窃 3 号房屋 (金额 = 9)，接着偷窃 5 号房屋 (金额 = 1)。
    - 偷窃到的最高金额 = 2 + 9 + 1 = 12 。
---
`python语法：动态规划(基础递推)`
```python
class Solution(object):
    def rob(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        # dp[i]代表在i号房累计可偷到最大的金额
            # 偷：最大金额为dp[i - 2] + nums[i]
            # 不偷：最大金额为dp[i - 1]
        # 初始化dp
        dp = []
        if(len(nums) == 0): return 0
        if(len(nums) == 1): return nums[-1]
        dp.append(nums[0])
        dp.append(max(nums[0], nums[1])) # dp[1]只是可以考虑偷2号房，并不是必须偷，需要考虑1-2的价值
        
        # dp递推
        # 可以理解为一开始就是双线，偷了1或2，后面以此为基础递推
        for i in range(2, len(nums)):
            dp.append(max(dp[i - 2] + nums[i], dp[i - 1])) # 偷or不偷
        return dp[-1]
```
