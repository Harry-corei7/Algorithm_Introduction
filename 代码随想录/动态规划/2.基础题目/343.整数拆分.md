`难度：中等`

- 给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。
- 示例 1:
  - 输入: 2
  - 输出: 1
  - 解释: 2 = 1 + 1, 1 × 1 = 1。

- 示例 2:
  - 输入: 10
  - 输出: 36
  - 解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。

- 说明: 你可以假设 n 不小于 2 且不大于 58。
---
> 动态规划思路

### (1) 确定dp数组（dp table）以及下标的含义

- dp[i]：分拆数字i，可以得到的最大乘积为dp[i]。

### (2) 确定递推公式(这一步不看题解真的不容易想到)

- 可以想 dp[i]最大乘积是怎么得到的呢？
- 其实可以从1遍历j，然后有两种渠道得到dp[i]，详细如下所示
- 假设对正整数 i 拆分出的第一个正整数是`j（1 <= j < i）`，则有以下两种方案：
  1) 将`i`拆分成`j`和`i−j`的和，且`i−j`**不再拆分**成多个正整数，此时的乘积是`j * (i-j)`
  2) 将`i`拆分成`j`和`i−j`的和，且`i−j`**继续拆分**成多个正整数，此时的乘积是`j * dp[i-j]`

### (3) dp的初始化

- 严格从dp[i]的定义来说，dp[0] dp[1] 就不应该初始化，也就是没有意义的数值。
- 只初始化`dp[2] = 1`，从dp[i]的定义来说，拆分数字2，得到的最大乘积是1

### (4) 确定遍历顺序

- 确定遍历顺序，先来看看递归公式：`dp[i] = max(dp[i], max((i - j) * j, dp[i - j] * j))`
- `dp[i]`是依靠`dp[i - j]`的状态，所以遍历i一定是从前向后遍历，先有dp[i - j]再有dp[i]。
- 所以遍历顺序代码如下所示：
```python
for i in range(3, n+1): # 从3递推到n
    for j in range(1, (i//2)+1): # 每次j从1拆分到i//2
         dp[i] = max(dp[i], max(j*(i-j), j*dp[i-j]))
```
- `遍历i，从3开始一直递推到n`
- `遍历j，从0开始无意义，从1开始递推到i//2，因为 j 和 (i-j) 是对称的，当 j > i/2 时，(i-j) 就会 < i/2，已经在之前遍历过了`
- `为什么要比较dp[i]`
  - j每一次遍历都会得出来一个dp[i]的值，遍历所有的j，会有依据这个j得出来的j个dp[i]
```
# 例如
# dp[4]=max(dp[4-j]j,(4-j)j)
# dp[4]=max(dp[4-1]*1,(4-1)*1)
# dp[4]=max(dp[4-2]*2,(4-2)*2)
# dp[4]=max(dp[4-3]*1,(4-3)*3)
# 每一次遍历j都会有dp[4]的更新，j遍历完取最大
```

`python语法：动态规划`
```python
class Solution(object):
    def integerBreak(self, n):
        """
        :type n: int
        :rtype: int
        """
        # 假设对正整数 i 拆分出的第一个正整数是 j（1 <= j < i），则有以下两种方案：
        # 1) 将 i 拆分成 j 和 i−j 的和，且 i−j 不再拆分成多个正整数，此时的乘积是 j * (i-j)
        # 2) 将 i 拆分成 j 和 i−j 的和，且 i−j 继续拆分成多个正整数，此时的乘积是 j * dp[i-j]
        
        dp = [0] * (n+1) # 首先创建一个长度为n+1的数组
        dp[2] = 1 # 下标0，1不可能有值，从下标2开始有值
        for i in range(3, n+1): # 从3递推到n
            for j in range(1, (i//2)+1): # 每次j从1拆分到i//2
                # i/2，得到结果是浮点数
                # print(10 / 2)   # 5.0（浮点数）
                # print(11 / 2)   # 5.5（浮点数）
                # print(9 / 2)    # 4.5（浮点数）
                # i//2，得到结果是向下取整的整数
                # print(-11 // 2)  # -6（整数）
                # print(11 // 2)   # 5（整数）
                # print(9 // 2)    # 4（整数）
                dp[i] = max(dp[i], max(j*(i-j), j*dp[i-j]))
                # 为什么要比较dp[i]：
                    # j每一次遍历都会得出来一个dp[i]的值，遍历所有的j，会有依据这个j得出来的j个dp[i]
                    # 例如
                    # dp[4]=max(dp[4-j]j,(4-j)j)
                    # dp[4]=max(dp[4-1]*1,(4-1)*1)
                    # dp[4]=max(dp[4-2]*2,(4-2)*2)
                    # dp[4]=max(dp[4-3]*1,(4-3)*3)
                    # 每一次遍历j都会有dp[4]的更新，j遍历完取最大
        return dp[n]
```
