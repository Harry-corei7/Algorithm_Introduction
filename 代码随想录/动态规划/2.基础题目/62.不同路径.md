`python语法：动态规划`
```python
class Solution(object):
    def uniquePaths(self, m, n):
        """
        :type m: int
        :type n: int
        :rtype: int
        """
        dp = [[0] * n for _ in range(m)]
        for i in range(0, n): dp[0][i] = 1
        for j in range(0, m): dp[j][0] = 1
        for i in range(1, n):
            for j in range(1, m):
                dp[j][i] = dp[j][i-1] + dp[j-1][i]
        return dp[m-1][n-1] 
```
