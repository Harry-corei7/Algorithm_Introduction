`难度：中等`

- 给你一个整数数组`coins`，表示不同面额的硬币；以及一个整数`amount`，表示总金额。
- 计算并返回可以凑成总金额所需的**最少的硬币个数**。如果没有任何一种硬币组合能组成总金额，返回`-1`。
- 你可以认为每种硬币的数量是无限的。

- 示例 1：
  - 输入：coins = [1, 2, 5], amount = 11
  - 输出：3 
  - 解释：11 = 5 + 5 + 1
- 示例 2：
  - 输入：coins = [2], amount = 3
  - 输出：-1
- 示例 3：
  - 输入：coins = [1], amount = 0
  - 输出：0
---
> 动态规划-完全背包(统计达到target的背包最小物品数量)思路

### (1) 利用float('intf')创建二维dp数组，初始化第一行/第一列 

- `float('intf')为python中的无穷大值`
  - 在第一行初始化时，并不是每一个`背包容量j`都有方法装满(出现了不能整除coins[0]的情况，没有方案) 
  - 面对`没有方案`的情况，这种情况不能作为下一行递推基础，我们需要**在核心递推公式**添加**判断**`!= float('inf')`
  - 代替用0初始化，导致创建二维dp数组元素都是0，无法辨别是否有方案，导致盲目在0基础上+1
  - 以下面例子证明：
    - coins = [2,3,4]
    - amount = 5
    - `?`**代表该情况没有方案**，值为`float('intf')`
    - 递推dp二维数组如下所示
```python
# [2,3,4] 5

0 ? 1 ? 2 ?
0 ? 1 1 ? 2
0 ? 1 1 1 2  
```

- `初始化dp二维数组第一行`
  - (1) 完全背包问题背景下，一个物品可以无限量装
  - (2) 在第一行，我们只考虑物品0，coins[0]
  - (3) 只有`背包容量/物品重量`**整除**，才算有方案，才能赋值
```python
for j in range(1, amount+1):
    if(j % coins[0] == 0):
        dp[0][j] = (j / coins[0]) 
```

- `初始化dp二维数组第一列`
  - (1) 有时初始化为`1`，有时初始化为`0`，本题初始化为`0`，题目要的时达到target的最小物品数量
```python
for i in range(len(coins)):
    dp[i][0] = 0
```

### (2) 核心递推公式

- `判断`：
  - (1) `j`需要大于等于`coins[i]`
  - (2) `dp[i][j - coins[i]]`这个前置位置的值，不能是`float('intf')`无方案 
```python
if(j >= coins[i] and dp[i][j - coins[0]] != float('intf')):
```
- `核心递推公式`：
  - 在`dp[i - 1][j]`和`dp[i][j - coins[0]] + 1`之间取**小的值**
```python
dp[i][j] = min(dp[i - 1][j], dp[i][j - coins[0]] + 1)
``` 

---
`python语法：动态规划-完全背包(统计达到target的背包最小物品数量)`
```python
class Solution(object):
    def coinChange(self, coins, amount):
        """
        :type coins: List[int]
        :type amount: int
        :rtype: int
        """
        dp = [[float('inf')]*(amount + 1) for _ in range(len(coins))]
        coins.sort()
        
        if(amount < 0): # 特殊情况：amount < 0, 直接不可能
            return -1
        if(amount == 0): # 特殊情况：amount = 0, 不是没方案所以=0，提前return 0
            return 0
        
        # 初始化
        for j in range(amount + 1):
            if(j % coins[0] == 0):
                dp[0][j] = (j/coins[0])
        for i in range(0, len(coins)):
            dp[i][0] = 0
        
        for i in range(1, len(coins)):
            for j in range(1, amount + 1):
                if(j >= coins[i] and dp[i][j - coins[i]] != float('inf')):
                    # 不能写dp[i][j] = dp[i][j - coins[i]] + 1
                    # 虽然有coins.sort()，还是需要比较，不一定就是dp[i][j - coins[i]] + 1最小
                    # 例外情况:
                    # [1, 20, 22] amount = 100
                    # dp[i-1][j] = 5
                    # dp[i][j - coins[i]] = 3+18(21)，再+1为22
                    dp[i][j] = min(dp[i][j - coins[i]] + 1, dp[i-1][j])
                else:
                    dp[i][j] = dp[i - 1][j]
                    
        
        if(dp[-1][-1] == float('inf')):
            return -1
        else:
            return dp[-1][-1]

        # 为什么要 != float('inf')
        # [2,3,4] 5

        # 0 - 1 - 2 -
        # 0 - 1 1 - 2
        # 0 - 1 1 1 2  
```
