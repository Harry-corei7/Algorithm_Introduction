`难度：中等`

- 给你一个字符串`s`和一个字符串列表`wordDict`作为字典。
- 如果可以利用字典中出现的一个或多个单词拼接出`s`则返回`true`。
- 注意：不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。

- 示例 1：
  - 输入: s = "leetcode", wordDict = ["leet", "code"]
  - 输出: true
  - 解释: 返回 true 因为 "leetcode" 可以由 "leet" 和 "code" 拼接成。
- 示例 2：
  - 输入: s = "applepenapple", wordDict = ["apple", "pen"]
  - 输出: true
  - 解释: 返回 true 因为 "applepenapple" 可以由 "apple" "pen" "apple" 拼接成。
  - 注意，你可以重复使用字典中的单词。
- 示例 3：
  - 输入: s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
  - 输出: false
---
`python语法：动态规划(排列问题-外背包内物品)`
```python
class Solution(object):
    def wordBreak(self, s, wordDict):
        """
        :type s: str
        :type wordDict: List[str]
        :rtype: bool
        """
        # 初始化dp，填充False(记录在这个长度能否被word排列组成)
        dp = [False] * (len(s)+1)
        dp[0] = True # 理解：空字符串可以成为排列组成s的一部分 / 实质：便于dp推导
        
        # 解决排列问题，有顺序(外层背包内层物品)，解决组合问题，无顺序(外层物品内层背包)
        
        # 外层：遍历背包容量
        for j in range(1, len(s)+1):
            # 内层：遍历物品
            for word in wordDict:
                if(j >= len(word)): # 背包(字符串长度)可以塞下当前物品
                    if(dp[j - len(word)] == True and word == s[j - len(word):j]):
                    # len(word)格子前的的状态为True 且 s[j-len(word):j]和word相等
                    # Python切片yyds
                        dp[j] = True # 有排列方案
                        break # 直接break
        
        return dp[-1]
```
