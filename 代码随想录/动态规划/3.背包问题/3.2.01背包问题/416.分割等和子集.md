`python语法：动态规划-01背包`
```python
class Solution(object):
    def canPartition(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """
        sum_nums = sum(nums)
        if(sum_nums % 2 == 1): return False # 如果数组的元素的值和，是奇数，则肯定无法分割成两个元素和相同的子集
        # 设置背包的容量，为数组元素和的一半，(能装满为True，否则为False)
        target = sum_nums/2
        # 初始化dp = [0] * target+1，初始化元素都为0
        dp = [0]*(target+1)
        
        # 一维数组dp遍历
        for i in range(len(nums)):
            for j in range(target, nums[i]-1, -1):
                dp[j] = max(dp[j], dp[j-nums[i]] + nums[i])
        
        if(dp[-1] == target): return True
        else: return False
```
