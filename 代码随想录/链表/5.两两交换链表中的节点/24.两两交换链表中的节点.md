`难度: 中等`

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

![image](https://github.com/user-attachments/assets/8346eb0f-f931-4782-a226-8621cee152b4)

输入：head = [1,2,3,4]
输出：[2,1,4,3]
示例 2：

输入：head = []
输出：[]
示例 3：

输入：head = [1]
输出：[1]

---
`java语法`
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode swapPairs(ListNode head) {
        // 创建一个虚拟头节点
        ListNode dumyhead = new ListNode(-1);
        dumyhead.next = head;

        // 三个地址变量
        ListNode firstNode, secondNode, temp;

        // 创建指针指向虚拟头节点
        ListNode cur = dumyhead;

        while(cur.next != null && cur.next.next != null){
            // 储存所有要用到的三个地址
            temp = cur.next.next.next;
            firstNode = cur.next;
            secondNode = cur.next.next;
            // 交换位置, 并重新连接temp
            cur.next = secondNode;
            cur.next.next = firstNode;
            firstNode.next = temp;
            // 移动指针, 便于下轮调换
            cur = firstNode;
        }
        return dumyhead.next;
    }
}
```
---
`python语法`
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def swapPairs(self, head):
        """
        :type head: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        # 设置虚拟头节点
        dumy_head = ListNode(-1)
        dumy_head.next = head
        # 设置三个指针，让两辆交换更为清晰
        # current, first_node, sec_node
        current = dumy_head
        while(current.next != None and current.next.next != None):
            # 赋值另外2个指针
            first_node = current.next
            sec_node = current.next.next
            # 进行交换
            current.next = sec_node
            first_node.next = sec_node.next
            sec_node.next = first_node
            # 移动current指针，便于进行下一轮
            current = first_node # 等效于current = current.next.next
        return dumy_head.next
```
- **这道题告诉，巩固了我们一个链表知识:**
- 当要对一个节点进行操作时，指针必须在这个节点的上一个节点
