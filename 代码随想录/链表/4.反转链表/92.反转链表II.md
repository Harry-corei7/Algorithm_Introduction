`难度：中等`

- 给你单链表的头指针`head`和两个整数`left`和`right`，其中`left <= right`。
- 请你反转从位置`left`到位置`right`的链表节点，返回**反转后的链表**。

- 示例 1：
  - 输入：head = [1,2,3,4,5], left = 2, right = 4
  - 输出：[1,4,3,2,5]

- 示例 2：
  - 输入：head = [5], left = 1, right = 1
  - 输出：[5]

`python语法：个人理解版`
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def reverseBetween(self, head, left, right):
        """
        :type head: Optional[ListNode]
        :type left: int
        :type right: int
        :rtype: Optional[ListNode]
        """
        cur_next_left = ListNode(-1) # 防止left就是头节点
        cur_next_left.next = head
        cur_pre_right = head
        # 将cur_next_left移动到left节点的上一个节点
        for i in range(left-1):
            cur_next_left = cur_next_left.next
        # 将cur_pre_right移动到right节点的下一个节点
        for i in range(right):
            cur_pre_right = cur_pre_right.next
        pre = cur_pre_right
        current = cur_next_left.next
        
        # 标准的反转链表
        while(current != cur_pre_right):
            temp = current.next
            current.next = pre
            pre = current
            current = temp
        cur_next_left.next = pre

        # left不是头节点，直接返回head / 是头节点，返回cur_next_left.next
        if(left > 1): return head
        else: return cur_next_left.next
```
