给你一个链表的头节点 head 和一个整数 val，请你删除链表中所有满足 Node.val == val的节点，并返回 新的头节点。

**难点: 头节点的判断和删除, 如何处理**
1) 直接在原链表的基础上, 将head往后移动一位, 相当于删除了头节点
2) 设置一个虚拟头节点, 将其连接在原链表上

![image](https://github.com/user-attachments/assets/9214b8ae-ecc1-4db8-84cf-1133cdac6da3)

- 输入：head = [1,2,6,3,4,5,6], val = 6
- 输出：[1,2,3,4,5]

示例 2：

- 输入：head = [], val = 1
- 输出：[]

示例 3：

- 输入：head = [7,7,7,7], val = 7
- 输出：[]

---

`方法1: 直接在原链表的基础上, 向后移动head`

`java语法`
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode removeElements(ListNode head, int val) {
        // 方法1: 在原链表基础上, 向后移动头节点
        // 我们需要的是一个val不是目标值的的head
        while(head != null && head.val == val){
            head = head.next;
        }
        // 若整个链表的val都是val, 直接删到head = null
        if(head == null){
            return head;
        }
        // 得到一个val不是val的head, 开始对接下来的节点进行判断
        ListNode current = head;
        while(current.next != null){
            if(current.next.val == val){
                current.next = current.next.next;
            } else {
                current = current.next;
            }
        }
        return head;
    }
}
```
`python语法`
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def removeElements(self, head, val):
        """
        :type head: Optional[ListNode]
        :type val: int
        :rtype: Optional[ListNode]
        """
        # 方法1: 在原链表基础上, 向后移动头节点
        # 特殊判断，链表不为空
        # 通过while循环，我们得到了一个"干净"的头节点
        while(head != None and head.val == val):
            # 如果head的下一个节点不为空，且"干净",移动头节点
            # 不用删除旧头节点，和现头节点的连接
            head = head.next
        # 如果整个链表都是val，head直接移动到None了
        if(head == None):
            return head
        # 设置指针，删除头节点后的val节点
        current = head
        while(current.next != None):
            if(current.next.val == val):
                current.next = current.next.next
            else:
                current = current.next
        return head
```

`方法2: 设置虚拟头节点 dummy_head, return dummy_head.next`

`java语法`
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode removeElements(ListNode head, int val) {
        // 方法2: 设置虚拟头节点，next指向原链表头节点
        // 难点: 不加一个新的头节点, 链表的原头节点无法删除
        ListNode dummy_head = new ListNode(-1);
        dummy_head.next = head;
        ListNode current = dummy_head;
        while(current.next != null){
            if(current.next.val == val){
                current.next = current.next.next;
            } else {
                current = current.next;
            }
        }
        // 注意不是返回head, 而是返回header.next
        return dummy_head.next;
    }
}
```
