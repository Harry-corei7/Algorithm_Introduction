- 给你一个链表的头节点`head`，判断链表中是否有环。
- 如果链表中有某个节点，可以通过连续跟踪`next`指针再次到达，则链表中存在环。
- 为了表示给定链表中的环，评测系统内部使用整数`pos`来表示链表尾连接到链表中的位置（索引从 0 开始）。
- `注意`：`pos`不作为参数进行传递。仅仅是为了标识链表的实际情况。

- 如果链表中存在环 ，则返回`true`。 否则，返回`false`。

---
`python语法：暴力解法`
```python
class Solution(object):
    def hasCycle(self, head):
        """
        :type head: ListNode
        :rtype: bool
        """
        dummy_head = ListNode(-1) # 虚拟头指针
        dummy_head.next = head # 虚拟头指针指向head，以便找环
        fast_cur = dummy_head  # 快指针
        step = 0 # 快指针步数
        
        while(fast_cur != None): # 出现None说明没环
            slow_cur = dummy_head # 快指针每走一步，慢指针都初始化在起点
            real_step = 0 # 慢指针步数
            fast_cur = fast_cur.next
            step += 1
            while(slow_cur != fast_cur): # 计算慢指针到达快指针位置的步数
                slow_cur = slow_cur.next
                real_step += 1
            if(real_step != step): # 步数不匹配：有环
                return True
        
        return False
```
