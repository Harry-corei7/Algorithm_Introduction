`难度：中等`

- 当且仅当每个相邻位数上的数字`x`和`y`满足`x <= y`时，我们称这个整数是单调递增的。
- 给定一个整数`n`，返回`小于或等于 n`的最大数字，且数字呈`单调递增`。

- 示例 1:
  - 输入: n = 10
  - 输出: 9
- 示例 2:
  - 输入: n = 1234
  - 输出: 1234
示例 3:
  - 输入: n = 332
  - 输出: 299
---
`python语法：贪心算法(个人理解版)`
```python
class Solution(object):
    def turnListtonum(self, num_list):
        num = 0
        for i in range(len(num_list)):
            num = num*10
            num += num_list[i]
        return num
    
    def monotoneIncreasingDigits(self, n):
        """
        :type n: int
        :rtype: int
        """
        num_str = str(n)
        num_n = int(num_str[len(num_str)-1])
        result = []
        result.append(num_n)

        for i in range(len(num_str)-2, -1, -1): # 关键：从后往前遍历
            if(int(num_str[i]) > num_n): 
                # 如果前一位比后一位大，排空result数字，全append(9)，然后append(前一位-1)               
                for j in range(len(result)):
                    result.pop(0)
                    result.append(9)
                num_n = int(num_str[i])-1
                result.append(num_n)
            else:
                num_n = int(num_str[i])
                result.append(num_n)

        result.reverse() # .reverse()返回None
        # .sort() .reverse() 这种方法直接使用就行

        # 注意：如果第一位是0，(由1-1得来)，则直接pop掉，结果数比num少一位
        if(result[0] == 0):
            result.pop(0)

        result_num = self.turnListtonum(result)
        return result_num
```
