`难度：中等`

- 给你一个整数数组`nums`，请你找出一个具有**最大和**的`连续子数组`（子数组最少包含一个元素），返回其最大和。
- 子数组是数组中的一个连续部分。

- 示例 1：
  - 输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
  - 输出：6
  - 解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
- 示例 2：
  - 输入：nums = [1]
  - 输出：1
- 示例 3：
  - 输入：nums = [5,4,-1,7,8]
  - 输出：23

---
`python语法：暴力解法(优化版)`
```python
class Solution(object):
    def maxSubArray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        result = nums[0]
        for i in range(0, len(nums)): # 设置起始位置
            count = 0
            for j in range(i, len(nums)): # 开始从起始位置进行加和
                count += nums[j]          # 使用"加和"，而不是sum()函数
                if(count >= result):
                    result = count
        return result
```
