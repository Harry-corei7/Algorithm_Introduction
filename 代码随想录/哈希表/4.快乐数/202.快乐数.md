编写一个算法来判断一个数 n 是不是快乐数。

「快乐数」 定义为：

对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。
然后重复这个过程直到这个数变为 1，也可能是 无限循环 但始终变不到 1。
如果这个过程 结果为 1，那么这个数就是快乐数。
如果 n 是 快乐数 就返回 true ；不是，则返回 false 。

 
示例 1：

输入：n = 19
输出：true

解释：
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1

示例 2：

输入：n = 2
输出：false

---
`java语法`
```java
class Solution {
    public boolean isHappy(int n) {
        Set<Integer> record = new HashSet<>();
        // 核心要点: 
        // (1) 当一个数不是欢乐数,达不到1,会陷入循环,一些数字反复出现
        // (2) 当数字反复出现时候,就需要哈希表了
        while(n != 1 && !record.contains(n)){
            record.add(n);
            n = getNextNumber(n);
        }
        if(record.contains(n)){
            return false;
        }
        
        return true;
    }

    public int getNextNumber(int n){
        int result =0;
        while(n != 0){
            int temp = n%10;
            result += temp*temp;
            n = n/10;
        }
        return result;
    }
}
```
---
`python语法`
```python
class Solution(object):
    def isHappy(self, n):
        """
        :type n: int
        :rtype: bool
        """
        # 核心要点:
        # (1) 当一个数不是快乐数的时候，会陷入循环，一些数字反复出现
        # (2) 为了判断数字是否反复出现，需要用到哈希表
        count = {}
        while(n != 1):
            if(n in count.keys()):
                return False
            else:
                count[n] = 1
            n = self.get_next_number(n)
        return True

    # 编写类方法，进行推导计算
    def get_next_number(self, num):
        """接收数字，返回其每一个数位的数字的平方和"""
        result = 0
        while(num != 0):
            temp = num % 10
            result += temp**2
            num = num/10 # 结果去除余数部分，等效于将num整体缩小10倍并去掉小数
        return result
```
