`难度: 中等`

给你一个由`n`个整数组成的数组`nums`，和一个目标值`target`。

请你找出并返回满足下述全部条件且不重复的四元组`[nums[a], nums[b], nums[c], nums[d]]`（若两个四元组元素一一对应，则认为两个四元组重复）：

- 0 <= a, b, c, d < n
- a、b、c 和 d 互不相同
- nums[a] + nums[b] + nums[c] + nums[d] == target

你可以按 任意顺序 返回答案 。

**通过18.四数之和, 结合15.三数之和, 可以推出五数之和, 六数之和, 乃至n数之和**

---
`java语法`
```java
class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        // 创建二位数组result
        List<List<Integer>> result = new ArrayList<>();
        // 给nums排序
        Arrays.sort(nums);

        for(int i = 0; i < nums.length; i++){
            // (1)第一次剪枝
            if(nums[i] >= 0 && nums[i] > target){
                break; // 这里的break等同于return result
            }
            // 对i进行去重, 当i>0后(第一轮后),开始对比i和上一轮是否相同
            if(i > 0 && nums[i] == nums[i-1]){
                continue;
            }

            for(int j = i + 1; j < nums.length; j++){
                // (2)第二次剪枝
                if(nums[j] + nums[i] >= 0 && nums[j] + nums[i] > target){
                    break; // 注意, 这里的break是返回上一层循环, 如果直接return result会有遗漏
                }
                // 对j进行去重, 当j>i+1后(第一轮后),开始对比j和上一轮是否相同
                if(j > i+1 && nums[j] == nums[j-1]){
                    continue;
                }
                // 双指针法
                int left = j+1, right = nums.length-1;
                while(left < right){
                    int sum = nums[i] + nums[j] + nums[left] + nums[right];

                    if(sum < target){
                        left++;
                    }
                    else if(sum > target){
                        right--;
                    }
                    else{
                        List<Integer> res = new ArrayList<>();
                        res.add(nums[i]);
                        res.add(nums[j]);
                        res.add(nums[left]);
                        res.add(nums[right]);
                        result.add(res);
                        // 对left, right去重
                        while(left < right && nums[left] == nums[left+1]){
                            left++;
                        }
                        while(left < right && nums[right] == nums[right-1]){
                            right--;
                        }
                        left++;
                        right--;
                    }
                }
            }
        }
        return result;
    }
}
```

`python语法`
```python
class Solution(object):
    def fourSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[List[int]]
        """
        nums.sort()
        result = []
        for a in range(len(nums)):
            # (1)第一次剪枝: 当a大于target，直接break
            # 注意: 这里比三数和多一个判断: 所有数都是正数时才成立。
            # 如果数组中有负数，即使 nums[a] > target，后面加上负数仍然可能等于目标值。
            if(nums[a] > 0 and nums[a] > target):
                break
            # 后对比: [a-1]和[a]
            if(a > 0 and nums[a] == nums[a-1]):
                continue

            for b in range(a+1, len(nums)):
                """这里对b的处理和a完全相等"""
                # (2)第二次剪枝: 当b大于target，直接break
                # 注意: 这里比三数和多一个判断: 所有数都是正数时才成立。
                # 如果数组中有负数，即使 nums[a] > target，后面加上负数仍然可能等于目标值。
                if(nums[a]+nums[b] > 0 and nums[b] > target):
                    break
                # 后对比: [b-1]和[b]，因为一开始的[b-1]是a+1，所以这里是b > a+1
                if(b > a+1 and nums[b] == nums[b-1]):
                    continue
                
                # 双指针法
                left = b+1
                right = len(nums)-1
                while(left < right):
                    if(nums[a]+nums[b]+nums[left]+nums[right] < target):
                        left += 1
                    elif(nums[a]+nums[b]+nums[left]+nums[right] > target):
                        right -= 1
                    else:
                        result_list = []
                        result_list.append(nums[a])
                        result_list.append(nums[b])
                        result_list.append(nums[left])
                        result_list.append(nums[right])
                        result.append(result_list)
                        # 对left，right去重
                        left += 1
                        right -= 1
                        while(left < right and nums[left] == nums[left-1]):
                            left += 1
                        while(left < right and nums[right] == nums[right+1]):
                            right -= 1
        return result
```
