给定两个数组`nums1`和`nums2`，返回 它们的 交集 。输出结果中的每个元素一定是 唯一 的。我们可以 不考虑输出结果的顺序 

示例 1：

输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]

示例 2：

输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
解释：[4,9] 也是可通过的

---
`java语法`
```java
class Solution {
    public int[] intersection(int[] nums1, int[] nums2) {
        ArrayList<Integer> List = new ArrayList<>();
        // 奇思妙想，类似给字母异位词int[26];
        // 数字就根据题目范围给一个可以包住全范围的
        // (1) 1 <= nums1.length, nums2.length <= 1000
        // (2) 最大可能, 每一位的元素都不一样, 共1000种元素
        int[] Count1 = new int[1001];
        int[] Count2 = new int[1001];

        for(int i = 0; i < nums1.length; i++){
            Count1[nums1[i]] = 1;
        }
        for(int i = 0; i < nums2.length; i++){
            Count2[nums2[i]] = 1;
        }
        // 用ArrayLsit存, 因为不知道结果有多少个
        for(int i = 0; i < 1001; i++){
            if(Count1[i] == 1 && Count2[i] == 1){
                List.add(i);
            }
        }
        // 根据ArrayList的大小创建int[], 并将其中的值放入int[]
        int[] result = new int[List.size()];

        for(int i = 0; i < List.size(); i++){
            result[i] = List.get(i);
        }
        return result;
    }
}
```

`python语法1: 用for i in range()遍历，等同于java语法`
```python
class Solution(object):
    def intersection(self, nums1, nums2):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :rtype: List[int]
        """
        count_1 = {}
        for i in range(1001):
            if(i in nums1):
                count_1[i] = count_1.get(i, 0) + 1
        count_2 = {}
        for i in range(1001):
            if(i in nums2):
                count_2[i] = count_2.get(i, 0) + 1
        result = []
        for i in range(1001):
            if(i in count_1.keys() and i in count_2.keys()):
                result.append(i)
        return result
```

`python语法1: 用for num in count遍历，python独有的针对元素进行遍历`
```python
class Solution(object):
    def intersection(self, nums1, nums2):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :rtype: List[int]
        """
        count_1 = {}
        for i in range(len(nums1)):
            count_1[nums1[i]] = count_1.get(nums1[i], 0) + 1
        count_2 = {}
        for i in range(len(nums2)):
            count_2[nums2[i]] = count_2.get(nums2[i], 0) + 1
        result = []
        for num in count_1:
            if num in count_2:
                result.append(num)
        return result
```
