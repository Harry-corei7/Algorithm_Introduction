给你两个整数数组 nums1 和 nums2 ，请你以数组形式返回两数组的交集。
- 返回结果中每个元素出现的次数，应与元素在两个数组中都出现的次数一致（如果出现次数不一致，则考虑取较小值）。
- 可以不考虑输出结果的顺序。

示例 1：

输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2,2]

示例 2:

输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[4,9]

---
`java语法`
`如果数组已排序, 双指针法`
`Arrays.sort方法的运用`

```java
import java.util.*;

class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        // 保证两个数组已排序
        Arrays.sort(nums1);
        Arrays.sort(nums2);
        // 建立List储存结果
        List<Integer> list = new ArrayList<>();
        int i = 0, j = 0;

        // 双指针遍历两个数组
        // 注意while里面的&&双条件语句 
        while (i < nums1.length && j < nums2.length) {
            if (nums1[i] == nums2[j]) {
                list.add(nums1[i]);
                i++;
                j++;
            } else if (nums1[i] < nums2[j]) {
                i++;
            } else {
                j++;
            }
        }

        // 转换为 int[] 返回
        int[] result = new int[list.size()];
        for (int k = 0; k < list.size(); k++) {
            result[k] = list.get(k);
        }
        return result;
    }
}
```

`我的简单方法`

```java
class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        // 延续两个数组的交集的解法
        int[] Count1 = new int[1001];
        int[] Count2 = new int[1001];
        // 不仅记录有没有, 而且记录出现多少次
        for(int i = 0; i < nums1.length; i++){
            Count1[nums1[i]]++;
        }
        for(int i = 0; i < nums2.length; i++){
            Count2[nums2[i]]++;
        }
        // 建立一个ArrayList来储存结果
        ArrayList<Integer> List = new ArrayList<>();
        // 注意: 设立一个temp储存Count1和Count2小的结果
        // 小的结果是多少就add多少个
        for(int i = 0; i < 1001; i++){
            if(Count1[i] >= 1 && Count2[i] >= 1){
                int temp;
                if(Count1[i] > Count2[i]){ 
                    temp = Count2[i]; 
                }
                else{ 
                    temp = Count1[i]; 
                }
                for(int j = 0; j < temp; j++){
                    List.add(i);
                }
            }
        }
        // 将ArrayList转换成数组
        int[] result = new int[List.size()];
        for(int i = 0; i < List.size(); i++){
            result[i] = List.get(i);
        }
        return result;
    }
}
```

`python语法`
```python
class Solution(object):
    def intersect(self, nums1, nums2):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :rtype: List[int]
        """
        count_1 = {}
        for i in range(len(nums1)):
            count_1[nums1[i]] = count_1.get(nums1[i], 0)+1
        count_2 = {}
        for i in range(len(nums2)):
            count_2[nums2[i]] = count_2.get(nums2[i], 0)+1
        result = []
        for key in count_1:
            if(key in count_2):
                # 如果两个字典都有这个key，则在两个字典的value中取小值
                num = min(count_1.get(key), count_2.get(key))
                for i in range(num):
                    result.append(key)
        return result
```
