`难度: 中等`

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

这道题目很好地考察了 **HashMap** 和 **ArrayList** 的运用

示例 1:
- 输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
- 输出: [["bat"],["nat","tan"],["ate","eat","tea"]]

示例 2:
- 输入: strs = [""]
- 输出: [[""]]

示例 3:
- 输入: strs = ["a"]
- 输出: [["a"]]

---
`java语法`
```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        // 题目要求返回互为字母异位词的词语的各自的列表
        // 使用HasMap<键, 值>: 键不能重复,值可以重复的特性
        // (1) 将Arrays.sort(temp)作为键, 所有字母异位词共享一个值
        // (2) 将list作为值, 到时候可以直接返回
        HashMap<String, ArrayList<String>> map = new HashMap<>();

        for(int i = 0; i < strs.length; i++){
            // 建立字符数组变量, 储存strs[i]
            char[] temp = strs[i].toCharArray();
            // 排序以便得到字母异位词的固定词
            Arrays.sort(temp);
            // 建立/取得对应的List<String>
            ArrayList<String> List = map.getOrDefault(new String(temp), new ArrayList<>());
            // 添加strs[i]元素
            List.add(strs[i]);
            // 新的List<String>代替旧的
            map.put(new String(temp), List);
        }
        // 返回map.values()
        return new ArrayList<>(map.values());
    }
}
```
---
`python语法`
```python
class Solution(object):
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        # 这道题确实很好地考察了对HasMap和动态数组地运用
        # 运用了HasMap键不能重复，值可以放数值，字符串，数组，字典等任意类型变量
        # (1) 键: temp.sort()，确定相同的字母异位词
        # (2) 值: 列表，直接append，到时候返回字典.values()，即为[字符串列表]列表
        result = {} # 构建hashmap
        for i in range(len(strs)):
            temp_list = list(strs[i])                    # 获得当前词的列表形式,便于排序获得唯一key
            temp_sorted_str = ''.join(sorted(temp_list)) # 排序,并将列表转化为字符串
            value_list = result.get(temp_sorted_str, []) # 字典的get方法,寻找key: temp_sorted_str, 找不到返回[]
            # 更新,key(temp_sorted_str)对应的value数组
            value_list.append(strs[i])                   
            result[temp_sorted_str] = value_list
        return result.values()
```
