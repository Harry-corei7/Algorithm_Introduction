`难度: 中等`

给你四个整数数组 nums1、nums2、nums3 和 nums4 ，数组长度都是 n ，请你计算有多少个元组 (i, j, k, l) 能满足：
- 0 <= i, j, k, l < n
- nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0
 
示例 1：

- 输入：nums1 = [1,2], nums2 = [-2,-1], nums3 = [-1,2], nums4 = [0,2]
- 输出：2

解释：

- 两个元组如下：
1. (0, 0, 0, 1) -> nums1[0] + nums2[0] + nums3[0] + nums4[1] = 1 + (-2) + (-1) + 2 = 0
2. (1, 1, 0, 0) -> nums1[1] + nums2[1] + nums3[0] + nums4[0] = 2 + (-1) + (-1) + 0 = 0

示例 2：

- 输入：nums1 = [0], nums2 = [0], nums3 = [0], nums4 = [0]
- 输出：1

---
`java语法`
```java
class Solution {
    public int fourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
        int result = 0;
        HashMap<Integer, Integer> map = new HashMap<>();
        // 先计算前两个数组的, 两数之和, 并统计出现次数
        for(int i : nums1){
            for(int j : nums2){
                int sum1 = i + j;
                map.put(sum1, map.getOrDefault(sum1, 0)+1);
            }
        }
        // 再计算后两个数组的, 两数之和的相反数, 并看看map里面对应的次数
        // 让result累加, 得到所有可能性的和
        for(int k : nums3){
            for(int l : nums4){
                int sum2 = - k - l;
                result += map.getOrDefault(sum2, 0);
            }
        }
        return result;
    }
}
```
---
`python语法`
```python
class Solution(object):
    def fourSumCount(self, nums1, nums2, nums3, nums4):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :type nums3: List[int]
        :type nums4: List[int]
        :rtype: int
        """
        count = {}
        # 先统计nums1和nums2的和，统计所有情况
        for i in nums1:
            for j in nums2:
                count[i + j] = count.get(i + j, 0)+1
        # 再遍历nums3和nums4的和的相反数，看看count里面符合的数的频率
        result = 0
        for k in nums3:
            for l in nums4:
                temp = -k-l
                # 有temp直接加频率
                # 没有，为防止报错，添加temp作为新key，但是频率=0，不影响
                result += count.get(temp, 0) 
        return result
```
