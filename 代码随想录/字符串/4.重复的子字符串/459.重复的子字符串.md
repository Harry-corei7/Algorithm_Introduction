给定一个非空的字符串 s ，检查是否可以通过由它的一个子串重复多次构成。

1) 示例 1:
- 输入: s = "abab"
- 输出: true
- 解释: 可由子串 "ab" 重复两次构成。
2) 示例 2:
- 输入: s = "aba"
- 输出: false
3) 示例 3:
- 输入: s = "abcabcabcabc"
- 输出: true
- 解释: 可由子串 "abc" 重复四次构成。 (或子串 "abcabc" 重复两次构成。)

提示：1 <= s.length <= 104

---

`老爸的穷举法`

```java
class Solution {
    public boolean repeatedSubstringPattern(String s) {
        // 穷举法:
        // (1) 从s的第1个字符开始, 不断增加子字符串的长度
        // (2) 判断s是否是由这个子字符串构成的

        for(int i = 1; i < s.length(); i++){
            String str = s.substring(0, i);
            // 判断特殊情况: 一旦str=s, 则返回false
            // 其实不用, str不可能 = s, 因为substring的特性
            if(str.length() == s.length()){
                return false;
            }
            // 判断特殊情况: 一旦s长度不能整除str长度, 则返回false
            else if(s.length() % i != 0){
                continue;
            }

            int m = 0;
            // 注意点: 在开始遍历s比对str前, 设置一个布尔型变量 = true
            boolean valid = true;

            for(int j = 0; j < s.length(); j++){
                if(s.charAt(j) != str.charAt(m)){
                    valid = false;
                    break;
                }
                m++;
                if(m == str.length()){
                    m = 0;
                }
            }
            if(valid){
                return true;
            }
        }
        return false;
    }
}
```
