给定 s 和 t 两个字符串，当它们分别被输入到空白的文本编辑器后，如果两者相等，返回 true 。# 代表退格字符。

注意：如果对空文本输入退格字符，文本继续为空。

示例 1：

输入：s = "ab#c", t = "ad#c"
输出：true
解释：s 和 t 都会变成 "ac"。
示例 2：

输入：s = "ab##", t = "c#d#"
输出：true
解释：s 和 t 都会变成 ""。
示例 3：

输入：s = "a#c", t = "b"
输出：false
解释：s 会变成 "c"，但 t 仍然是 "b"。

---
`java语法`
```java
class Solution {
    public boolean backspaceCompare(String s, String t) {
        int i = s.length()-1, j = t.length()-1;
        int skipS = 0, skipT= 0;

        while(i >= 0 || j >= 0){
            while(i >= 0) {
                // 处理s的退格
                if(s.charAt(i) == '#'){ skipS++; i--; } 
                else if(skipS > 0){ skipS--; i--; }
                else{ break; }
            }

            while(j >= 0) {
                // 处理t的退格
                if(t.charAt(j) == '#'){ skipT++; j--; } 
                else if(skipT > 0){ skipT--; j--; }
                else{ break; }
            }

            // 在移除 "#" 和 "被#删除的char" 的间隙
            // (1) 两条字符串都没走到头(-1)，检查相等
            if(i >= 0 && j >= 0 && (t.charAt(j) != s.charAt(i)) ){
                return false;
            }
            
            // (2) 如果一条字符串没走到头(-1), 另一条还没走到头(-1)
            if((i >= 0) != (j >= 0)){
                return false; // 说明另一条有多余字符
            }
            i--;
            j--;
        }
        return true;
    }
}
```

`python语法`
```python
class Solution(object):
    def backspaceCompare(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        i, s_skip = len(s) - 1, 0
        j, t_skip = len(t) - 1, 0
        while(i >= 0 or j >= 0):
            # 处理s里面的#
            while(i >= 0):
                if(s[i] == "#"):
                    s_skip += 1
                    i -= 1
                elif(s_skip > 0):
                    s_skip -= 1
                    i -= 1
                else:
                    break
            # 处理j里面的#
            while(j >= 0):
                if(t[j] == "#"):
                    t_skip += 1
                    j -= 1
                elif(t_skip > 0):
                    t_skip -= 1
                    j -= 1
                else:
                    break
            # 处理#之余
            # (1) 确保没有走到头(-1), 进行比较
            if(s[i] != t[j] and i >= 0 and j <= 0):
                return False
            # (2) 有走到头(-1), 但另一个还没到头
            if(i < 0 or j < 0):
                if(i < 0 and j >= 0):
                    return False
                if(j < 0 and i >= 0):
                    return False
                if(i < 0 and j < 0):
                    return True
            # (2) 可以写成 if((i < 0) != (j < 0))
            i -= 1
            j -= 1
        return True
```
