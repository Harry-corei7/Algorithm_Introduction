**栈 Stack（后进先出 LIFO）**

![image](https://github.com/user-attachments/assets/a784236d-843c-447a-9ddb-eb406e9b832d)

`1.常规方法操作:`

|操作|说明
|--- |---
|`push()`| 入栈（放到栈顶）
|`pop()`| 出栈（移除栈顶）
|`peek()`| 查看栈顶元素但不移除
|`empty()`| 判断栈是否为空
|`size()`| 查看栈中元素的多少

`2.Java 中的栈类:`

```java
Stack<T> stack = new Stack<>();
```

`3.示例代码`

```java
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        Stack<Integer> stack = new Stack<>();

        // 入栈
        stack.push(10);
        stack.push(20);
        stack.push(30);
        System.out.println("当前栈: " + stack); // [10, 20, 30]

        // 查看栈顶
        System.out.println("栈顶元素: " + stack.peek()); // 30

        // 出栈
        int popped = stack.pop(); // 移除30
        System.out.println("弹出元素: " + popped); // 30
        System.out.println("栈状态: " + stack); // [10, 20]

        // 判空
        System.out.println("栈是否为空: " + stack.empty()); // false
    }
}
```
