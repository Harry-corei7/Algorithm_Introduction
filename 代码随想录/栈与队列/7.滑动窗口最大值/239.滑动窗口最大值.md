`难度: 困难`

给你一个整数数组`nums`，有一个大小为`k`的滑动窗口从数组的最左侧移动到数组的最右侧。

你只可以看到在滑动窗口内的`k`个数字。滑动窗口每次只向右移动一位。

返回 滑动窗口中的最大值 。

1) 示例 1：

- 输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
- 输出：[3,3,5,5,6,7]
- 解释：

滑动窗口的位置                   最大值
- [1  3  -1] -3  5  3  6  7       3
- 1 [3  -1  -3] 5  3  6  7        3
- 1  3 [-1  -3  5] 3  6  7        5
- 1  3  -1 [-3  5  3] 6  7        5
- 1  3  -1  -3 [5  3  6] 7        6
- 1  3  -1  -3  5 [3  6  7]       7

2) 示例 2：

- 输入：nums = [1], k = 1
- 输出：[1]

---
`我的穷举法`

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        Queue<Integer> queue = new LinkedList<>();
        int[] res = new int[nums.length-k+1];
        
        for(int i = 0; i < nums.length; i++){
            if(queue.size() < k){
                queue.offer(nums[i]);
            }
            if(queue.size() == k){
                int maxnum = nums[i];
                for(int j = 0; j < k-1; j++){
                    maxnum = Math.max(maxnum, Math.max(nums[i-j-1], nums[i-j]));
                }
                res[i-k+1] = maxnum;
                int num1 = queue.poll();
            }
        }
        return res;
    }
}
```
