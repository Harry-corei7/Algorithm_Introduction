`难度: 困难`

给你一个整数数组`nums`，有一个大小为`k`的滑动窗口从数组的最左侧移动到数组的最右侧。

你只可以看到在滑动窗口内的`k`个数字。滑动窗口每次只向右移动一位。

返回 滑动窗口中的最大值 。

1) 示例 1：

- 输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
- 输出：[3,3,5,5,6,7]
- 解释：

滑动窗口的位置              ----------   最大值
- [1  3  -1] -3  5  3  6  7  ------       3
- 1 [3  -1  -3] 5  3  6  7   ------       3
- 1  3 [-1  -3  5] 3  6  7   ------       5
- 1  3  -1 [-3  5  3] 6  7   ------       5
- 1  3  -1  -3 [5  3  6] 7   ------       6
- 1  3  -1  -3  5 [3  6  7]  ------       7

2) 示例 2：

- 输入：nums = [1], k = 1
- 输出：[1]

---
`我们的进阶方法+(i > index + k - 1)`

`java语法`
```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        int[] res = new int[nums.length-k+1];
        int temp = 0;
        int index = 0;
        
        for(int i = k-1; i < nums.length; i++){
            // 首轮循环
            if(i == k-1 || i > (index + k - 1)){
                temp = nums[i];
                index = i;
                for(int j = 1; j <= k-1; j++){
                    if(nums[i-j] > temp){
                        temp = nums[i-j];
                        index = i-j;
                    }
                    // 记录最大值temp和index
                }
            }
            // 接下来，先对比每一个新的一位数字
            if(nums[i] > temp){
                temp = nums[i];
                index = i;
            }
            res[i-k+1] = temp;
        }
        return res;
    }
}
```
`python语法`
```python
class Solution(object):
    def maxSlidingWindow(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: List[int]
        """
        res = []
        index = 0
        temp = 0
        for i in range(k-1, len(nums)):
            if(i == k-1 or i > index+k-1): # 首轮or到达index临界值
                temp = nums[i]
                index = i
                for j in range(k-1):
                    if(nums[i-j-1] > temp): # 往前对比k-1个
                        temp = nums[i-j-1]
                        index = i-j-1 # 记录最大值和index
            if(nums[i] > temp): # 接下来，先对比每一个新的一位数字
                temp = nums[i]
                index = i
            res.append(temp)
        return res
```

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        Queue<Integer> queue = new LinkedList<>();
        int[] res = new int[nums.length-k+1];
        
        for(int i = 0; i < nums.length; i++){
            if(queue.size() < k){
                queue.offer(nums[i]);
            }
            if(queue.size() == k){ // 当队列元素数量达到k时，res数组记录当前队列中的最大值
                int maxnum = nums[i];
                for(int j = 0; j < k-1; j++){
                    maxnum = Math.max(maxnum, Math.max(nums[i-j-1], nums[i-j]));
                } // 找出当前队列的最大值
                res[i-k+1] = maxnum;
                int num1 = queue.poll();
            }
        }
        return res;
    }
}
```

`python语法`
```python
class Solution(object):
    def maxSlidingWindow(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: List[int]
        """
        left = 0
        result = []
        for right in range(len(nums)):
            if(right - left +1 == k):
                temp = max(nums[left: right+1]) # 找出当前队列的最大值
                result.append(temp)
                left += 1
        return result
```
