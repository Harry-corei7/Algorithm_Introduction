请你仅使用两个队列实现一个后入先出（LIFO）的栈，并支持普通栈的全部四种操作（push、top、pop 和 empty）。

实现 MyStack 类：

- void push(int x) 将元素 x 压入栈顶。
- int pop() 移除并返回栈顶元素。
- int top() 返回栈顶元素。
- boolean empty() 如果栈是空的，返回 true ；否则，返回 false 。

`注意：`

- 你只能使用队列的标准操作 —— 也就是`push to back`、`peek/pop from front`、`size`和`is empty`这些操作。
- 你所使用的语言也许不支持队列。 你可以使用`list（列表）`或者`deque（双端队列）`来模拟一个`队列`,只要是标准的队列操作即可。
 
`示例：`

- 输入：
- ["MyStack", "push", "push", "top", "pop", "empty"]
- [[], [1], [2], [], [], []]
- 输出：
- [null, null, null, 2, 2, false]

`解释：`

- MyStack myStack = new MyStack();
- myStack.push(1);
- myStack.push(2);
- myStack.top(); // 返回 2
- myStack.pop(); // 返回 2
- myStack.empty(); // 返回 False

`java语法`
```java
class MyStack {
    Queue<Integer> queue_1 = new LinkedList<>();
    Queue<Integer> queue_2 = new LinkedList<>();
    Queue<Integer> full, empty;

    public MyStack() {

    }
    
    public void push(int x) {
        if(queue_1.isEmpty() && queue_2.isEmpty()){
            full = queue_1;
            empty = queue_2;
        } else if(queue_1.isEmpty()){
            empty = queue_1;
            full = queue_2;
        } else{
            full = queue_1;
            empty = queue_2;
        }
        full.offer(x);
    }
    
    public int pop() {
        if(queue_1.isEmpty()){
            empty = queue_1;
            full = queue_2;
        } else{
            full = queue_1;
            empty = queue_2;
        }
        int size = full.size();
        for(int i = 0; i < size-1; i++){
            int k = full.poll();
            empty.offer(k);
        }
        int num = full.poll();
        return num;
    }
    
    public int top() {
        if(queue_1.isEmpty()){
            empty = queue_1;
            full = queue_2;
        } else{
            full = queue_1;
            empty = queue_2;
        }
        int size = full.size();
        for(int i = 0; i < size-1; i++){
            int k = full.poll();
            empty.offer(k);
        }
        int num = full.poll();
        empty.offer(num);
        return num;
    }
    
    public boolean empty() {
        if(queue_1.isEmpty()){
            empty = queue_1;
            full = queue_2;
        } else{
            full = queue_1;
            empty = queue_2;
        }
        return full.isEmpty();
    }
}
```

```python
class MyStack(object):

    def __init__(self):
        self.queue_1 = []
        self.queue_2 = []


    def push(self, x):
        """
        :type x: int
        :rtype: None
        """
        self.queue_1.append(x)
        

    def pop(self):
        """
        :rtype: int
        """
        while(len(self.queue_1) > 1):
            temp = self.queue_1.pop(0)
            self.queue_2.append(temp)
        res = self.queue_1.pop(0)
        while(len(self.queue_2) != 0):
            temp = self.queue_2.pop(0)
            self.queue_1.append(temp)
        return res


    def top(self):
        """
        :rtype: int
        """
        while(len(self.queue_1) > 1):
            temp = self.queue_1.pop(0)
            self.queue_2.append(temp)
        res = self.queue_1.pop(0)
        self.queue_2.append(res)
        while(len(self.queue_2) != 0):
            temp = self.queue_2.pop(0)
            self.queue_1.append(temp)
        return res

        

    def empty(self):
        """
        :rtype: bool
        """
        if(len(self.queue_1) == 0):
            return True
        else:
            return False
```
