给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

- 左括号必须用相同类型的右括号闭合。
- 左括号必须以正确的顺序闭合。
- 每个右括号都有一个对应的相同类型的左括号。
 
1) 示例 1：
- 输入：s = "()"
- 输出：true

2) 示例 2：
- 输入：s = "()[]{}"
- 输出：true

3) 示例 3：
- 输入：s = "(]"
- 输出：false

4) 示例 4：
- 输入：s = "([])"
- 输出：true

---

```java
class Solution {
    public boolean isValid(String s) {
        // 创建一个栈, 完美解决
        Stack<Character> stack = new Stack<>();
        for(char c : s.toCharArray()){
            if(c == ')' && !stack.empty() && stack.peek() == '('){
                stack.pop();
            }
            else if(c == ']' && !stack.empty() && stack.peek() == '['){
                stack.pop();
            }
            else if(c == '}' && !stack.empty() && stack.peek() == '{'){
                stack.pop();
            }
            else{
                stack.push(c);
            }
        }
        return stack.empty();
    }
}
```
