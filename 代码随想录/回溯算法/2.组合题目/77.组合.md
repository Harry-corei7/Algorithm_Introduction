`难度：中等`

- 给定两个整数`n`和`k`，返回范围`[1, n]`中所有可能的`k`个数的组合。
- 你可以按`任何顺序`返回答案。

- 示例 1：
- 输入：n = 4, k = 2
- 输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]

- 示例 2：
- 输入：n = 1, k = 1
- 输出：[[1]]

### 回溯法三部曲
> 1.递归函数的返回值以及参数
- 在这里要定义`两个全局变量`，一个用来存放符合条件单一结果，一个用来存放符合条件结果的集合。
```python
result:List[int] = []; // 存放符合条件结果的集合
path:List[List[int]] = []; // 用来存放符合条件结果
```

`python语法：递归回溯法`
```python
class Solution(object):
    def backtracking(self, n, k, startIndex, result, path):
        if(len(path) == k):
            result.append(path[:])
            return
        for i in range(startIndex, n+1): # 控制树的横向遍历
            path.append(i) # 处理节点
            self.backtracking(n, k, i+1, result, path) # 递归：控制树的纵向遍历，注意下一层搜索要从i+1开始
            path.pop() # 回溯，撤销处理的节点


    def combine(self, n, k):
        """
        :type n: int
        :type k: int
        :rtype: List[List[int]]
        """
        result = [] # 二位数组，存放集合结果
        path = [] # 存放节点
        self.backtracking(n, k, 1, result, path)
        return result
```
