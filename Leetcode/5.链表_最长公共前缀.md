**注意: substring(0, n) 返回第一个字符串的前 n 个字符。**

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1：

输入：strs = ["flower","flow","flight"]
输出："fl"
示例 2：

输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。

**我的方法**

---

```java
class Solution {
    public String longestCommonPrefix(String[] strs) {
        
        if(strs.length < 2){
            return strs[0];
        } // 防止strs =["a"]的特殊情况，需要返回["a"]
        if(strs[0].length() == 0){
            return "";
        } // 防止strs =["","",""]的特殊情况，需要返回[""]

        // 定义一个储存公约数的数组
        int[] a = new int[strs.length];
        // 获取前缀相同的公约数
        for(int i = 1; i < strs.length; i++){
            int m = 0;
            for(int j=0; j<strs[i].length() && j<strs[0].length(); j++){   
                if(strs[i].charAt(j) ==  strs[0].charAt(j)){
                    m++;
                }else{
                    break;
                }
            }
            a[i-1] = m;
        }
        // 找出最小公约数
        int n = a[0];
        for(int i = 1; i < strs.length-1 ; i++){
            if(a[i]<n){
                n = a[i]; 
            }
        }
        // 利用substring返回最小公约数的字符串
        return(strs[0].substring(0,n));
    }
}
```
