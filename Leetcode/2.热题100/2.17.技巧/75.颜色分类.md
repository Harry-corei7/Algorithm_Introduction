`难度：中等`

- 给定一个包含红色、白色和蓝色、共`n`个元素的数组`nums`，**原地**对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。
- 我们使用整数`0、 1 和 2`分别表示`红色、白色和蓝色`。
- 必须在不使用库内置的`sort`函数的情况下解决这个问题。

- 示例 1：
  - 输入：nums = [2,0,2,1,1,0]
  - 输出：[0,0,1,1,2,2]
- 示例 2：
  - 输入：nums = [2,0,1]
  - 输出：[0,1,2]

---

`python：哈希表`
```python
class Solution(object):
    def sortColors(self, nums):
        """
        :type nums: List[int]
        :rtype: None Do not return anything, modify nums in-place instead.
        """
        # 创建字典收集"颜色"数量
        nums_dict = {
            0:0,
            1:0,
            2:0 
        }
        for i in range(len(nums)):
            nums_dict[nums[i]] = nums_dict.get(nums[i], 0) + 1

        k = 0 # 创建k作为总下标
        # 开始给原数组nums赋值
        while(nums_dict[0] > 0):
            for i in range(k, nums_dict[0]):
                nums[i] = 0
                k += 1
                nums_dict[0] -= 1
        while(nums_dict[1] > 0):
            for i in range(k, k+nums_dict[1]):
                nums[i] = 1
                k += 1
                nums_dict[1] -= 1
        while(nums_dict[2] > 0):
            for i in range(k, k+nums_dict[2]):
                nums[i] = 2
                nums_dict[2] -= 1
    
```
