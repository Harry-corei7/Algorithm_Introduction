- 给定一个非负整数`numRows`，生成「杨辉三角」的前`numRows`行。
- 在「杨辉三角」中，每个数是它左上方和右上方的数的和。

<img width="452" height="327" alt="image" src="https://github.com/user-attachments/assets/67841015-63f4-4315-bbcb-20bed5d42706" />

- 示例 1:
  - 输入: numRows = 5
  - 输出: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
- 示例 2:
  - 输入: numRows = 1
  - 输出: [[1]]
---
`python语法：个人理解版`
```python
class Solution(object):
    def generate(self, numRows):
        """
        :type numRows: int
        :rtype: List[List[int]]
        """
        res = [[1]]
        for i in range(numRows - 1):
            num = [1]
            if(len(res) > 1):
                for k in range(1, len(res[-1])):
                    num.append(res[-1][k - 1] + res[-1][k])
            num.append(1)
            res.append(num)
        return res
```
