`难度：中等`

- 给你一个整数数组 nums 和一个整数 k ，请你统计并返回 该数组中和为 k 的子数组的个数 。
- 子数组是数组中元素的连续非空序列。

- 示例 1：
  - 输入：nums = [1,1,1], k = 2
  - 输出：2
- 示例 2：
  - 输入：nums = [1,2,3], k = 3
  - 输出：2
**提示：**
  - `1 <= nums.length <= 2 * 104`
  - `-1000 <= nums[i] <= 1000`

---
`python语法：哈希表+前缀和`
```python
class Solution(object):
    def subarraySum(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: int
        """
        # 核心思路：哈希表记录前缀和
        # (1) 我们要找满足 pre[i] - pre[j-1] == k 的 (j, i) 子数组数。
        # (2) 起点 j 真的不唯一！ 因为数组里可以有负数和 0，导致前面不同的位置，算出来的前缀和居然是一样的。
        # 等价于：在遍历到 i 时，
            # 前缀和为 pre[i] - k 的个数 = (j, i) 子数组数


        res = 0
        # 哈希表：{前缀和:出现次数}
        prefix_count = {0:1}
        # 记录数组[0:当前元素]的总和
        cur_num = 0
        for i in range(len(nums)):
            cur_num += nums[i]
            # 查找是否存在前缀和等于 cur_sum - k
            if(cur_num - k in prefix_count):
                res += prefix_count[cur_num - k]
            # 记录当前数组的前缀和
            prefix_count[cur_num] = prefix_count.get(cur_num, 0) + 1
        return res
```
