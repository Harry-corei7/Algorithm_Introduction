- 给定一个**无重复元素**的**有序**整数数组`nums`。
- 区间`[a,b]`是从`a`到`b`（包含）的所有整数的集合。
- 返回**恰好覆盖数组中所有数字**的**最小有序**区间范围列表。
- 也就是说，`nums`的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个区间但不属于`nums`的数字`x`。
- 列表中的每个区间范围 [a,b] 应该按如下格式输出：
  - "a->b" ，如果 a != b
  - "a" ，如果 a == b

- 示例 1：
  - 输入：nums = [0,1,2,4,5,7]
  - 输出：["0->2","4->5","7"]
  - 解释：区间范围是：
    - [0,2] --> "0->2"
    - [4,5] --> "4->5"
    - [7,7] --> "7"
- 示例 2：
  - 输入：nums = [0,2,3,4,6,8,9]
  - 输出：["0","2->4","6","8->9"]
  - 解释：区间范围是：
    - [0,0] --> "0"
    - [2,4] --> "2->4"
    - [6,6] --> "6"
    - [8,9] --> "8->9"
---
`python语法`
```python
class Solution(object):
    def summaryRanges(self, nums):
        """
        :type nums: List[int]
        :rtype: List[str]
        """
        # 特殊情况
        if(len(nums) == 0): return []
        # 设置二维数组，每一个区间独占一个一维数组
        res = [[nums[0]]]
        suf = nums[0]
        index = 0
        for i in range(1, len(nums)):
            if(nums[i] == suf + 1):
                if(len(res[index]) == 1): # 如果没加"->"，加上
                    res[index].append("->")
                if(len(res[index]) == 3): # 如果已经有了右端，更新
                    res[index].pop(-1)
                res[index].append(nums[i])
                suf += 1
            else:
                res.append([nums[i]])
                suf = nums[i]
                index += 1
        
        # 将二维数组转化为一位字符串数组
        for i in range(len(res)):
            for j in range(len(res[i])):
                res[i][j] = str(res[i][j])
            res[i] = "".join(res[i]) # 将字符数组的字符串元素连接成一个字符串
        return res
```
