- 将两个升序链表合并为一个新的`升序`链表并返回。
- 新链表是通过拼接给定的两个链表的所有节点组成的。

<img width="662" height="302" alt="image" src="https://github.com/user-attachments/assets/dda9f66a-edd2-4167-b5ee-560ccb3c0182" />

- 示例 1：
  - 输入：l1 = [1,2,4], l2 = [1,3,4]
  - 输出：[1,1,2,3,4,4]
- 示例 2：
  - 输入：l1 = [], l2 = []
  - 输出：[]
- 示例 3：
  - 输入：l1 = [], l2 = [0]
  - 输出：[0]

---
`python语法`
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def mergeTwoLists(self, list1, list2):
        """
        :type list1: Optional[ListNode]
        :type list2: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        res = ListNode(-1)
        cur_res = res
        cur_1 = list1
        cur_2 = list2
        while(cur_1 != None or cur_2 != None):
            cur_res.next = ListNode(-1)
            cur_res = cur_res.next
            if(cur_1 != None and cur_2 != None):
                if(cur_1.val <= cur_2.val):
                    cur_res.val = cur_1.val
                    cur_1 = cur_1.next
                else:
                    cur_res.val = cur_2.val
                    cur_2 = cur_2.next
            else:
                if(cur_1 != None):
                    cur_res.val = cur_1.val
                    cur_1 = cur_1.next
                else:
                    cur_res.val = cur_2.val
                    cur_2 = cur_2.next
        return res.next
```
