- 给定一种规律`pattern`和一个字符串`s`，判断`s`是否遵循相同的规律。
- 这里的**遵循**指完全匹配，例如，`pattern`里的每个字母和字符串`s`中的每个非空单词之间存在着双向连接的对应规律。
- 具体来说：
  - `pattern`中的每个字母都**恰好**映射到`s`中的一个唯一单词。
  - `s`中的每个唯一单词都**恰好**映射到`pattern`中的一个字母。
  - 没有两个字母映射到同一个单词，也没有两个单词映射到同一个字母。

- 示例1:
  - 输入: pattern = "abba", s = "dog cat cat dog"
  - 输出: true
- 示例 2:
  - 输入:pattern = "abba", s = "dog cat cat fish"
  - 输出: false
- 示例 3:
  - 输入: pattern = "aaaa", s = "dog cat cat dog"
  - 输出: false
---
`python语法`
```python
class Solution(object):
    # 将带有" "的字符串转化为列表
    def stringToWordList(self, s):
        str_list = [""]
        j = 0
        for i in range(0, len(s)):
            if(s[i] != " "):
                str_list[j] += s[i]
            else:
                str_list.append("")
                j += 1
        return str_list

    def wordPattern(self, pattern, s):
        """
        :type pattern: str
        :type s: str
        :rtype: bool
        """
        str_list = self.stringToWordList(s)
        if(len(pattern) != len(str_list)): return False # 如果len不相等，直接return False
        word_dict = {}
        for i in range(len(pattern)):
            if(pattern[i] not in word_dict and str_list[i] not in word_dict.values()):
                word_dict[pattern[i]] = str_list[i]
            elif(pattern[i] in word_dict and str_list[i] in word_dict.values()):
                if(word_dict[pattern[i]] != str_list[i]): return False
            else:
                return False
        return True
```
