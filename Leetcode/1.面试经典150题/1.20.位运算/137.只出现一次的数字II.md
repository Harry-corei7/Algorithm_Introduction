`难度：中等`

- 给你一个整数数组`nums`，除某个元素仅出现**一次**外，其余每个元素都恰出现**三次**。
- 请你找出并返回那个只出现了一次的元素。
- 你必须设计并实现线性时间复杂度的算法且使用常数级空间来解决此问题。

- 示例 1：
    - 输入：nums = [2,2,3,2]
    - 输出：3
- 示例 2：
    - 输入：nums = [0,1,0,1,0,1,99]
    - 输出：99
---
### 补充知识：根据32位整数的32位补码的运算法则

- 注意题目的范围：`-2^31 <= nums[i] <= 2^31 - 1`
    - 即题目数字范围为，`32位整数`
- **正数的32位补码运算：**
    - 根据2次方对32位补码的换算(0的补码是32个0)：
        - `2^0` = `1`，也就是cnt[0] = 1，cnt[31]到cnt[1]数组这31位置都是0
        - `2^1` = `2`，也就是cnt[1] = 1，cnt[30]到cnt[1]数组这30位置都是0，cnt[0]=0
        - ......
        - `2^31 - 1`，也就是cnt[31] = 0，cnt[30]到cnt[0]都是 1
- **负数的32位补码运算：**
    - 负数的补码等于其绝对值的原码取反再加 1
        - 比如：`-1`的补码 = `1`的补码取反 + 1，结果为32个1
        - 原因：-1 + 1，机器计算补码刚刚好 = 10000(32个0)，而33位置的1直接(进位丢失)，结果为32个0 
        - `-2^31`，也就是cnt[31] = 1，cnt[30]到cnt[0]都是 0
- 因此，负数的cnt[31] = 1，而非负数(正数和0)的cnt[31] = 0

---
`python语法：32位补码`
```python
class Solution(object):
    def singleNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        # 1.统计二进制每一个数的1出现的次数
        cnt = [0] * 32  # 长度为 32 的列表，对应 32 个比特位
        for num in nums:
            for j in range(32):
                # 取出 num 的二进制的第 j 位，并加到计数中
                cnt[j] += (num >> j) & 1 # num的32位补码向右移动j位，最后一位和1位运算
        
        # 2.根据统计的每一个数字的32位二进制的累加，重建结果
        res = 0
        for i in range(32):
            if(cnt[i] % 3 == 1): # 该位的值不是3的倍数，说明出现一次的数在这个32位二进制的位置的值为 1
                res = res | (1 << i) # 1的32位补码向左移动j位(右边不断补充0)，和res的32位补码的第 i 位置位运算
                # 将 res 的32位二进制的第 i 位置设为 1
        
         # 3. 处理符号：Python 整数无位数限制，若第 31 位为 1，需转换为负数
        if res & (1 << 31): # 检查最高位（符号位）
            res -= (1 << 32) # 转换成有符号 32 位整数的负数形式
        
        return res
```
