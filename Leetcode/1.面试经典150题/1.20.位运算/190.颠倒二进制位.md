- 颠倒给定的`32`位有符号整数的二进制位。

- 示例 1：
  - 输入：n = 43261596
  - 输出：964176192
  - 解释：
|整数	|二进制 |
|---|---|
| 43261596 |00000010100101000001111010011100|
|964176192 |00111001011110000010100101000000|

- 示例 2：
  - 输入：n = 2147483644
  - 输出：1073741822
  - 解释：
|整数	|二进制 |
|---|---|
|2147483644	|01111111111111111111111111111100|
|1073741822	|00111111111111111111111111111110|

---
`python语法`
```python
class Solution(object):
    def twoToTenList(self, num):
        tenList = []
        while(num != 0):
            tenList.append(num % 2) # 直接创建目标整数的32位二进制
            num = num // 2
        while(len(tenList) < 32):
            tenList.append(0)
        return tenList

    def reverseBits(self, n):
        """
        :type n: int
        :rtype: int
        """
        res_list = self.twoToTenList(n)
        while(len(res_list) >= 1 and res_list[0] == 0): # 将开头的"无效"的0去除(由颠倒导致的)
            res_list.pop(0) # 保留一位数字，预防n = 0，0本身的32位全是0
        j = 0
        res = 0
        for i in range(len(res_list)-1, -1, -1):
            res += res_list[i] * (2**j)
            j += 1
        return res
        
```
