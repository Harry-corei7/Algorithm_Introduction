- 给你两个二进制字符串`a`和`b`，以二进制字符串的形式返回它们的和。

- 示例 1：
  - 输入:a = "11", b = "1"
  - 输出："100"
- 示例 2：
  - 输入：a = "1010", b = "1011"
  - 输出："10101"
---
`python语法`
```python
class Solution(object):
    # 将2进制转换为10进制
    def twoToTen(self, num):
        res = 0
        j = 0
        for i in range(len(num) - 1, -1, -1):
            res += int(num[i]) * (2 ** j)
            j += 1
        return res


    def addBinary(self, a, b):
        """
        :type a: str
        :type b: str
        :rtype: str
        """
        a_10 = self.twoToTen(a)
        b_10 = self.twoToTen(b)
        res_10 = a_10 + b_10
        # 将10进制转化为2进制
        if(res_10 == 0): return "0" # 特殊情况
        res_2 = ""
        while(res_10 != 0):
            res_2 = str(res_10 % 2) + res_2
            res_10 = res_10 // 2
        return res_2
```
