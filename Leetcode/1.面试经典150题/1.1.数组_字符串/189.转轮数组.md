`难度：中等`

- 给定一个整数数组`nums`，将数组中的元素向右轮转`k`个位置，其中`k`是非负数。

- 示例 1:
  - 输入: nums = [1,2,3,4,5,6,7], k = 3
  - 输出: [5,6,7,1,2,3,4]
  - 解释:
    - 向右轮转 1 步: [7,1,2,3,4,5,6]
    - 向右轮转 2 步: [6,7,1,2,3,4,5]
    - 向右轮转 3 步: [5,6,7,1,2,3,4]

- 示例 2:
  - 输入：nums = [-1,-100,3,99], k = 2
  - 输出：[3,99,-1,-100]
  - 解释: 
    - 向右轮转 1 步: [99,-1,-100,3]
    - 向右轮转 2 步: [3,99,-1,-100]
---
`python语法(个人理解版)`
```python
class Solution(object):
    def rotate(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: None Do not return anything, modify nums in-place instead.
        """
        # 超时，需要判断k和len(nums)
        while(k > len(nums)): # 剪枝(1)
            k = k % len(nums)
        if(k == len(nums)): # 剪枝(2)
            return nums
        else:
            if(k <= len(nums)//2 ):
                for i in range(k):
                    num = nums.pop(-1)
                    nums.insert(0, num)
            else:
                for i in range(len(nums) - k):
                    num = nums.pop(0)
                    nums.append(num)
            return nums
```
