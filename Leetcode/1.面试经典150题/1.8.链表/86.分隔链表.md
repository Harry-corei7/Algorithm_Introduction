`难度：中等`

- 给你一个链表的头节点`head`和一个特定值`x`，请你对链表进行分隔，使得所有**小于**`x`的节点都出现在**大于或等于**`x`的节点之前。
- 你应当**保留**两个分区中每个节点的初始相对位置。

- 示例 1：
  - 输入：head = [1,4,3,2,5,2], x = 3
  - 输出：[1,2,2,4,3,5]

- 示例 2：
  - 输入：head = [2,1], x = 2
  - 输出：[1,2]

`python语法：个人理解版`
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def partition(self, head, x):
        """
        :type head: Optional[ListNode]
        :type x: int
        :rtype: Optional[ListNode]
        """
        dummy_head_original = ListNode(-1) # 创建虚拟头节点便于删除原链表
        dummy_head_original.next = head
        dummy_head_new = ListNode(-1) # 创建虚拟头节点便于新建链表
        pre = dummy_head_original
        cur_original = head
        cur_new = dummy_head_new      
        while(cur_original != None):
            if(cur_original.val < x):
                # 将节点从原链表中删除
                pre.next = cur_original.next
                cur_original.next = None
                # 新链表添加节点
                cur_new.next = cur_original
                # 指针复位
                cur_original = pre.next
                cur_new = cur_new.next
            else:
                cur_original = cur_original.next
                pre = pre.next
        # 将 <x 的新链表与旧链表连接
        cur_new.next = dummy_head_original.next
        return dummy_head_new.next
```
