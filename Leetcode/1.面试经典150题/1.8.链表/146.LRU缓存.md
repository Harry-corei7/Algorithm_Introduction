`难度：中等`

- 请你设计并实现一个满足**LRU (最近最少使用) 缓存**约束的数据结构。
- 实现`LRUCache`类：
  - `LRUCache(int capacity)`以**正整数**作为容量`capacity`初始化`LRU 缓存`
  - `int get(int key)`如果关键字`key`存在于缓存中，则返回关键字的值，否则返回`-1`。
  - `void put(int key, int value)`如果关键字`key`已经存在，则变更其数据值`value`；
  - 如果不存在，则向缓存中插入该组`key-value`。如果插入操作导致关键字数量超过`capacity`，则应该**逐出**最久未使用的关键字。
  - 函数`get`和`put`必须以`O(1)`的平均时间复杂度运行。

- 示例：
  - 输入
    - ["LRUCache", "put", "put", "get", "put", "get", "put", "get", "get", "get"]
    - [[2], [1, 1], [2, 2], [1], [3, 3], [2], [4, 4], [1], [3], [4]]
  - 输出
    - [null, null, null, 1, null, -1, null, -1, 3, 4]
  - 解释
    - LRUCache lRUCache = new LRUCache(2);
    - lRUCache.put(1, 1); // 缓存是 {1=1}
    - lRUCache.put(2, 2); // 缓存是 {1=1, 2=2}
    - lRUCache.get(1);    // 返回 1
    - lRUCache.put(3, 3); // 该操作会使得关键字 2 作废，缓存是 {1=1, 3=3}
    - lRUCache.get(2);    // 返回 -1 (未找到)
    - lRUCache.put(4, 4); // 该操作会使得关键字 1 作废，缓存是 {4=4, 3=3}
    - lRUCache.get(1);    // 返回 -1 (未找到)
    - lRUCache.get(3);    // 返回 3
    - lRUCache.get(4);    // 返回 4

`python语法：双向链表-字典，字典：{key(键): 双向链表的节点(值)}`
```python
class ListNode: # 创建双向链表
    def __init__(self, key = 0, value = 0):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None

class LRUCache(object):
    def __init__(self, capacity):
        """
        :type capacity: int
        """
        self.cache = {} # cache表示存储器
        self.capacity = capacity

        # 虚拟头尾节点，简化边界处理
        self.dumy_head = ListNode()
        self.dumy_tail = ListNode()
        self.dumy_head.next = self.dumy_tail
        self.dumy_tail.prev = self.dumy_head

    
    def _remove(self, node):
        """从链表中删除节点"""
        pre = node.prev
        nxt = node.next
        pre.next = nxt
        nxt.prev = pre
    
    def _add_to_head(self, node):
        """将一个新的头节点添加进双向链表头"""
        node.prev = self.dumy_head
        node.next = self.dumy_head.next
        self.dumy_head.next.prev = node
        self.dumy_head.next = node
    
    def _move_to_head(self, node):
        """将节点移动到头部(最近使用)"""
        self._remove(node)
        self._add_to_head(node)
        

    def get(self, key):
        """
        :type key: int
        :rtype: int
        """
        if(key in self.cache):
            node = self.cache[key]
            self._move_to_head(node) # (最近使用) 
            return node.value
        else:
            return -1


    def put(self, key, value):
        """
        :type key: int
        :type value: int
        :rtype: None
        """
        if(key in self.cache):
            node = self.cache[key]
            self._move_to_head(node) # (最近使用) 
            node.value = value
        else:
            if(len(self.cache) == self.capacity): # 若达到容量阈值
                del_key = self.dumy_tail.prev.key
                del self.cache[del_key]
                self._remove(self.dumy_tail.prev)
            # 添加新节点
            new_node = ListNode(key, value)
            self._add_to_head(new_node)
            self.cache[key] = new_node
```

`python语法：数组-字典，个人理解法`
```python
class LRUCache(object):

    def __init__(self, capacity):
        """
        :type capacity: int
        """
        self.key = [] # 队列，记录key的顺序
        self.dict = {} # 记录key-value
        self.capacity = capacity


    def get(self, key):
        """
        :type key: int
        :rtype: int
        """
        if(self.dict.get(key, -1) != -1): # 若key存在，访问表示使用，更新队列中key的顺序
            self.key.remove(key)
            self.key.append(key)
            return self.dict[key]
        return -1


    def put(self, key, value):
        """
        :type key: int
        :type value: int
        :rtype: None
        """
        if(key in self.dict): # 若key存在，更改表示使用，更新队列中key的顺序
            self.key.remove(key)
        else:
            if(len(self.key) == self.capacity):
                del self.dict[self.key.pop(0)]
        self.key.append(key)
        self.dict[key] = value
```
