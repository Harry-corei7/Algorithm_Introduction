`难度：中等`

- 实现`pow(x, n)`，即计算`x`的整数`n`次幂函数（即，`x^n`）。

- 示例 1：
  - 输入：x = 2.00000, n = 10
  - 输出：1024.00000
- 示例 2：
  - 输入：x = 2.10000, n = 3
  - 输出：9.26100
- 示例 3：
  - 输入：x = 2.00000, n = -2
  - 输出：0.25000
  - 解释：2-2 = 1/22 = 1/4 = 0.25
---
`python语法：折半计算法-递归`
```python
class Solution(object):
    def pow(self, x, n):
        # 需要用到递归
        if(n == 0): return 1
        if(n == 1): return x
        # 如果n为偶数，可以拆分的话
        if(n % 2 == 0):
            res = self.pow(x, n/2)
            return res * res # 拆分后，获得结果更块
        # 如果n为奇数，不可拆分
        else:
            res = self.pow(x, n//2) # 整数除法，向下取整
            return res * res * x # 两个整数除法(n // 2)，刚刚好只差一个1次方，所以*x补上1

    def myPow(self, x, n):
        """
        :type x: float
        :type n: int
        :rtype: float
        """
        if(n >= 0):
            return self.pow(x, n)
        else:
            x = 1/x
            return self.pow(x, -n)
```
