`难度：中等`

- 给定一个整数`n`，返回`n!`结果中尾随零的数量。
- 提示`n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1`

- 示例 1：
  - 输入：n = 3
  - 输出：0
  - 解释：3! = 6 ，不含尾随 0
- 示例 2：
  - 输入：n = 5
  - 输出：1
  - 解释：5! = 120 ，有一个尾随 0
- 示例 3：
  - 输入：n = 0
  - 输出：0
---
`python语法(这题考得真是数学)`
```python
class Solution(object):
    def trailingZeroes(self, n):
        """
        :type n: int
        :rtype: int
        """
        # 因为在阶乘中，2的因子远多于5的因子
        # 所以我们只需要计算5的因子，即可得出10的因子个数，即0的数量
        # 第1次：n //= 5 = 5  → count = 5  (5,10,15,20,25各贡献1个5)
        # 第2次：n //= 5 = 1  → count = 6  (25额外贡献1个5)
        # 第3次：n //= 5 = 0  → 结束
        # 结果：6个0 (25! = 15511210043330985984000000)
        zero_num = 0
        while(n > 0):
            n = n // 5
            zero_num += n
        return zero_num
```
