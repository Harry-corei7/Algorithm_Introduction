**一、StringBuilder —— 可变字符串**

`1.概念`

- `StringBuilder`是`Java`中提供的一个可变字符串类，位于`java.lang`包中。
- 作用：用于频繁拼接、插入、删除字符的场景，避免`String`不可变带来的效率低下问题。

`2. 常用方法`
|方法|含义
|---|---
|`append(String s)`|添加内容到末尾
|`insert(int offset, String s)`|在指定位置插入
|`delete(int start, int end)`|删除区间内的字符
|`reverse()`|反转整个字符串
|`setCharAt(int index, char c)`|修改指定位置的字符
|`charAt(int index)`|读取指定位置的字符
|`length()`|返回当前长度
|`toString()`|转换为`String`类型

`3. 示例代码`

```java
public class StringBuilderDemo {
    public static void main(String[] args) {
        // 创建一个空的 StringBuilder
        StringBuilder sb = new StringBuilder();

        // 1. append(): 追加内容到末尾
        sb.append("Hello");
        sb.append(" ");
        sb.append("World");
        System.out.println("1. append: " + sb); // 输出：Hello World

        // 2. insert(): 在指定位置插入字符串
        sb.insert(5, ", Java");
        System.out.println("2. insert: " + sb); // 输出：Hello, Java World

        // 3. delete(): 删除某个区间内的字符
        sb.delete(5, 11); // 删除 ", Java"
        System.out.println("3. delete: " + sb); // 输出：Hello World

        // 4. setCharAt(): 修改某个字符
        sb.setCharAt(6, 'w');
        System.out.println("4. setCharAt: " + sb); // 输出：Hello world

        // 5. charAt(): 读取某个字符
        char ch = sb.charAt(0);
        System.out.println("5. charAt: First char = " + ch); // 输出：H

        // 6. length(): 获取当前长度
        int len = sb.length();
        System.out.println("6. length: " + len); // 输出：11

        // 7. reverse(): 反转整个字符串
        sb.reverse();
        System.out.println("7. reverse: " + sb); // 输出：dlrow olleH

        // 8. toString(): 转换为 String 类型
        String result = sb.toString();
        System.out.println("8. toString: " + result); // 输出：dlrow olleH
    }
}

```
