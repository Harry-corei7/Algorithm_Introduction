**一、什么是 StringBuilder**

`1.概念`

- StringBuilder 是 Java 中提供的一个可变字符串类，位于 java.lang 包中。
- 作用：用于频繁拼接、插入、删除字符的场景，避免 String 不可变带来的效率低下问题。

`示例代码`

```java
StringBuilder sb = new StringBuilder();
sb.append("Hello");
sb.append(" ");
sb.append("World");
System.out.println(sb);           // 输出：Hello World
System.out.println(sb.toString()); // 等价输出：Hello World
```
