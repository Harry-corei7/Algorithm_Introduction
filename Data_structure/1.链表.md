**1.链表的基本概念**
- 链表是由多个 ListNode 节点组成的线性数据结构。每个节点包含两部分：
1)数据域（val）：存储节点的值。
2)指针域（next）：指向下一个节点。(单链表)
- 链表的最后一个节点的 next 为 null，表示链表的结束。

ListNode 类的结构
```java
class ListNode {
    int val;         // 节点的值
    ListNode next;   // 指向下一个节点的引用
    // 这里并没有创建ListNode对象,只是创建了一个next变量,用于储存下一个节点的地址(类似指针)
    // next默认为null

    // 构造函数
    ListNode(int x) { 
        val = x; 
    }
}
```

---

**2.类的成员**

1)int val：
- 表示节点存储的数据。这里假设节点存储的是整数类型（int），但可以根据需要改为其他类型（如 String、double 等）。

2)ListNode next：
- 是一个指向下一个节点的引用（指针）。
- 如果当前节点是链表的最后一个节点，则 next 为 null。

3) 构造函数 ListNode(int x)：
- 用于初始化一个链表节点。
- 参数 x 是节点的值，赋值给 val。
- next 默认为 null，表示当前节点没有指向下一个节点。
