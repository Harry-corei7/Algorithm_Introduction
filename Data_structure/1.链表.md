**1.链表的基本概念**
1) 链表是由多个 ListNode 节点组成的线性数据结构。每个节点包含两部分：
- 数据域（val）：存储节点的值。
- 指针域（next）：指向下一个节点。(单链表)

2) 链表的最后一个节点的 next 为 null，表示链表的结束。

,$p_n$

ListNode 类的结构
```java
class ListNode {
    int val;         // 节点的值
    ListNode next;   // 指向下一个节点的引用
    // 这里并没有创建ListNode对象,只是创建了一个next变量,用于储存下一个节点的地址(类似指针)
    // next默认为null

    // 构造函数
    ListNode(int x) { 
        val = x; 
    }
}
```

---

**2.类的成员**

1) int val：
- 表示节点存储的数据。这里假设节点存储的是整数类型（int），但可以根据需要改为其他类型（如 String、double 等）。

2) ListNode next：
- 是一个指向下一个节点的引用（指针）。
- 如果当前节点是链表的最后一个节点，则 next 为 null。

3) 构造函数 ListNode(int x)：
- 用于初始化一个链表节点。
- 参数 x 是节点的值，赋值给 val。
- next 默认为 null，表示当前节点没有指向下一个节点。

---

**3.链表的相关操作**

1) 创建一个链表

```java
ListNode head = new ListNode(1);
head.next = new ListNode(2);
head.next.next = new ListNode(3);
head.next.next.next = new ListNode(4);
```

2) 遍历链表

```java
ListNode current = head; // 从头节点开始
while (current != null) {
    System.out.print(current.val + " "); // 打印当前节点的值
    current = current.next; // 移动到下一个节点
}
```
```
1 2 3 4
```

3) 在链表末尾添加节点

```java
ListNode newNode = new ListNode(5); // 创建新节点
ListNode current = head;
while (current.next != null) {
    current = current.next; // 找到最后一个节点
}
current.next = newNode; // 将新节点添加到末尾
```

4) 删除链表中的节点

```java
// 删除值为 3 的节点
ListNode current = head;
while (current.next != null) {
    if (current.next.val == 3) {
        current.next = current.next.next; // 跳过值为 3 的节点
        break;
    }
    current = current.next;
}
```

5) 查找链表的长度

```java
int length = 0;
ListNode current = head;
while (current != null) {
    length++;
    current = current.next;
}
System.out.println("链表长度: " + length);
```

6) 将一个数组转化为单链表

```java
class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

// 输入一个数组，转换为一条单链表
ListNode createLinkedList(int[] arr) {
    if (arr == null || arr.length == 0) {
        return null;
    }
    ListNode head = new ListNode(arr[0]);
    ListNode cur = head;
    for (int i = 1; i < arr.length; i++) {
        cur.next = new ListNode(arr[i]);
        cur = cur.next;
    }
    return head;
}
```
