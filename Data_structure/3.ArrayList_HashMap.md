**一、ArrayList —— 动态数组**

`1. 概念`
- ArrayList 是 Java 提供的一个**可变长度的数组（也称为动态数组）**。
- 存储元素是**有序的、可重复的**；
- 元素的索引从**0 开始**。

`2. 底层原理`
- ArrayList 底层使用一个 数组 Object[] 来存储元素；
- 当元素个数超过当前数组容量时，会进行 扩容（通常扩为原容量的 1.5 倍）；
- 扩容涉及到数组复制，性能相对较低。

`3. 常用方法`
|方法|含义
|---|---
|`add(E e)`|末尾添加元素
|`add(int index, E e)`|指定位置插入
|`get(int index)`|获取指定索引的元素
|`set(int index, E e)`|替换指定位置元素
|`remove(int index)`|删除指定索引元素
|`size()`|返回元素数量
|`clear()`|清空所有元素
|`contains(Object o)`|判断是否包含指定元素

`4. 示例代码`

```java
import java.util.ArrayList;

ArrayList<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
list.add("cherry");
System.out.println(list.get(1)); // 输出: banana
list.set(1, "blueberry");
list.remove(0);
System.out.println(list); // 输出: [blueberry, cherry]
```

`5. 复杂ArrayList类型声明List<List<Integer>>`

这是一个泛型接口类型声明，表示：
- 外层是一个 List（例如：ArrayList）
- 内层每个元素也是一个 List<Integer>，即整型元素的列表

```java
[
  [1, 2, 3],
  [4, 5],
  [6]
]
```

**二、HashMap —— 键对值容器**

`1. 概念`
- HashMap 是 Java 中常用的**键值对（key-value）映射容器**；
- 允许 null 值，最多一个键为**null**；
- **键无序，不能重复**，**值可以重复**。

`2. 底层原理`
- 使用 数组 + 链表 + 红黑树 的结构：
  - 哈希冲突不多时，使用链表；
  - 冲突多了（链表长度超过 8），自动转换为红黑树，提高查询效率；
- 哈希函数（hashCode()）用于确定元素的存储位置；
- 负载因子默认为 0.75，超过阈值就扩容为原来的 2 倍。

`3. 常用方法`
|方法|含义
|---|---
|`put(K key, V value)`|添加或更新键值对
|`get(K key)`|获取对应值
|`containsKey(K key)`|是否存在某键
|`containsValue(V value)`|是否存在某值
|`remove(K key)`|删除指定键及其值
|`keySet()`|获取所有键集合
|`values()`|获取所有值集合
|`entrySet()`|获取键值对集合

`4. 示例代码`
```java
import java.util.HashMap;

HashMap<String, Integer> map = new HashMap<>();
map.put("apple", 3);
map.put("banana", 5);
map.put("apple", 7); // 覆盖旧值
System.out.println(map.get("apple")); // 输出: 7
System.out.println(map.containsKey("banana")); // true
map.remove("banana");
System.out.println(map); // 输出: {apple=7}
```

`5. HashMap.getOrDefault() 方法详解`
1) 方法作用
- 从`HashMap`中获取指定键`key`对应的值。如果这个`key`不存在，则返回你提供的`defaultValue`。

2) 示例代码
```java
import java.util.HashMap;

public class Main {
    public static void main(String[] args) {
        HashMap<String, Integer> map = new HashMap<>();
        map.put("apple", 3);

        int a = map.getOrDefault("apple", 0);    // key 存在，返回 3
        int b = map.getOrDefault("banana", 0);   // key 不存在，返回默认值 0

        System.out.println("apple = " + a);  // 输出: apple = 3
        System.out.println("banana = " + b); // 输出: banana = 0
    }
}

```

3) 比如用于统计字符频率：
```java
for (char c : s.toCharArray()) {
    map.put(c, map.getOrDefault(c, 0) + 1);
}

```
4) 分组
```java
List<String> list = map.getOrDefault(key, new ArrayList<>());
list.add(word);
map.put(key, list);
```
