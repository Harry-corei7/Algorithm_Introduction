**栈 Stack（后进先出 LIFO）**

![image](https://github.com/user-attachments/assets/a784236d-843c-447a-9ddb-eb406e9b832d)

`1.常规方法操作:`

|操作|说明
|--- |---
|`push()`| 入栈（放到栈顶）
|`pop()`| 出栈（移除栈顶）
|`peek()`| 查看栈顶元素但不移除
|`empty()`| 判断栈是否为空
|`size()`| 查看栈中元素的多少

`2.Java 中的栈类:`

```java
Stack<T> stack = new Stack<>();
```

`3.示例代码`

```java
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        Stack<Integer> stack = new Stack<>();

        // 入栈
        stack.push(10);
        stack.push(20);
        stack.push(30);
        System.out.println("当前栈: " + stack); // [10, 20, 30]

        // 查看栈顶
        System.out.println("栈顶元素: " + stack.peek()); // 30

        // 出栈
        int popped = stack.pop(); // 移除30
        System.out.println("弹出元素: " + popped); // 30
        System.out.println("栈状态: " + stack); // [10, 20]

        // 判空
        System.out.println("栈是否为空: " + stack.empty()); // false
    }
}
```

**队列 Queue（先进先出 FIFO）**

![image](https://github.com/user-attachments/assets/893c29b2-5050-4915-a99e-7f9244f1dc2d)

`1.常规操作`

|操作|说明
|---|---
|offer()| 入队（添加到尾部）
|poll()| 出队（移除头部元素）
|peek()| 查看队首元素但不移除
|isEmpty()| 判断队列是否为空
|size()| 查看队列中元素的多少

`2.Java 中常用队列实现：`
- 接口：Queue<T>（不能直接 new）
- 常用实现类：LinkedList<T>、ArrayDeque<T>、PriorityQueue<T> 等

`3.示例代码（用 LinkedList 实现队列）`

```java
import java.util.LinkedList;
import java.util.Queue;

public class QueueExample {
    public static void main(String[] args) {
        Queue<String> queue = new LinkedList<>();

        // 入队
        queue.offer("A");
        queue.offer("B");
        queue.offer("C");
        System.out.println("当前队列: " + queue); // [A, B, C]

        // 查看队首
        System.out.println("队首元素: " + queue.peek()); // A

        // 出队
        String removed = queue.poll();
        System.out.println("出队元素: " + removed); // A
        System.out.println("当前队列: " + queue); // [B, C]

        // 判空
        System.out.println("队列是否为空: " + queue.isEmpty()); // false
    }
}
```
