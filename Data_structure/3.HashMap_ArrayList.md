**一、ArrayList —— 动态数组**

`1. 概念`
- ArrayList 是 Java 提供的一个 可变长度的数组（也称为动态数组）。
- 存储元素是 有序的、可重复的；
- 元素的索引从 0 开始。

`2. 底层原理`
- ArrayList 底层使用一个 数组 Object[] 来存储元素；
- 当元素个数超过当前数组容量时，会进行 扩容（通常扩为原容量的 1.5 倍）；
- 扩容涉及到数组复制，性能相对较低。

`3. 常用方法`
|方法|含义
|---|---
|`add(E e)`|末尾添加元素
|`add(int index, E e)`|指定位置插入
|`get(int index)`|获取指定索引的元素
|`set(int index, E e)`|替换指定位置元素
|`remove(int index)`|删除指定索引元素
|`size()`|返回元素数量
|`clear()`|清空所有元素
|`contains(Object o)`|判断是否包含指定元素

`4. 示例代码`

```java
import java.util.ArrayList;

ArrayList<String> list = new ArrayList<>();
list.add("apple");
list.add("banana");
list.add("cherry");
System.out.println(list.get(1)); // 输出: banana
list.set(1, "blueberry");
list.remove(0);
System.out.println(list); // 输出: [blueberry, cherry]
```
